January #

January 1st, Wednesday #

转眼又到新的一年；就算按照最保守的算法，我也27岁了。出国五年，我的主要精力都放在自我的成长上。而到了去年秋冬，家庭的责任迎面而来，让我猝不及防。我也因此决定，把寻找伴侣正式提上日程。事业、家人、婚姻，人生的大事总是躲不开的。

既然事业的目标是找教职，当务之急是多发优质论文。之前的进度很小；虽然去年秋冬有所加速，也是远远不够。科研的推进，是接下来的重中之重。读书修身，则托个人网站而渐成体系，当再接再厉。

亲人们的身体，其实五年来早就小毛病不断；只是我一直没有去重视，也没有付出足够的耐心去改变状况。夏天老爸来上海看我，隐约记得是想顺便看个什么病，我也依然抛在脑后。直到下半年，老爸、姥爷、刘叔都突然被查出癌症，我才终于傻了眼。

刘叔已经去世；姥爷年纪太大，也只能静养，等待日子到来。唯独老爸，我无论如何也希望他能康复。以目前的情况来看，直肠和肝脏上的恶性肿瘤尚可手术去除，但肺部和淋巴上是否有转移，情况不明。我内心当然希望并无转移，但必须做好一切准备。现在要做的事，一是要和爸妈、二姨夫、张叔经常沟通，保证信息明确畅通；二是抓紧报名新药实验，争取最好的医疗条件；三是返美后广泛地收集信息，并且去医院咨询，争取凭借现有国内的材料，拿到美国医生的建议治疗方案；四是要备好材料，催促爸妈尽快办理好美国签证，以备出国看病的需要。

除此之外，今年起，我自己也要每年做两次以上的全面身体检查，及早发现和处理问题。要多了解医疗保险相关信息，在有可能的时候，尽早给自己和家人购买优质的健康保险。

出国五年，和梦莹、慧唅谈了两段正式的恋爱。去年下半年，则是和雨婕“网约SM”，胡闹了四个月。听到老爸得癌症的那天，忽然觉得自己不该再虚耗时光，而是该认真找个人生伴侣。爱情、事业、家人；到了这个年纪，我越来越需要有人一同分享和分担。

这次返美后，我打算一面接触朋友圈子内可能的对象，一面在“CoffeeMeetsBagle”上尝试约会。在这个过程中，重要的是明确自己想要的是什么。如果总是吹毛求疵地不满足，难有好的结果。

上月我下了决心，不再做任何“网约SM”相关的事，希望给老爸的身体祈福。明天起，我决心不再看任何形式的动画漫画，直到找到我内心认同的伴侣。这并非是和上帝、神明去做什么交换，只是想体现自己的一种决心，想对自己做一种提醒。好像高考前吃素，是一种正心诚意的方式。

January 2nd, Thursday #

到延吉后几次出门，都有些着凉。前天早上起来，发现自己有些感冒；到前天晚上，已经头痛得厉害。昨天早上起床时觉得好了些，但晚上又开始难受；一量体温，竟然38.8度了，赶紧去医院验了血 (确认是细菌感染)、打了吊瓶和屁股针。打过针，情况立刻就好些。今天午饭后又打了一次，到晚上，虽然喉咙还很疼，身上已经恢复正常了。明天上午再打一针，大致上就可以痊愈。

因为感冒好转，就计划明天动身去吉林。和家里人都打招呼问过之后，定了中午的动车票；打算等到达吉林，就直接打车到三叔三婶家里。本来计划后天就回通化，但宏伟说后天到家，因此犹豫是不是要等他一天。归心似箭，只想早点见到爸妈，希望一切顺利。

老妈说，叔婶他们还不知道老爸的事；要不要告诉，她在犹豫，但倾向于不说。如果不说，我一想到他们会来要这要那，或是絮叨些有的没的，就气不打一处来。但如果说了，又担心这些人瞎着急，抄起电话狂轰滥炸，说些不知轻重深潜的话。想到这里，心理有些烦躁，也有些疲惫。

如果说了，就得说清楚：(1) 不准让我爸知道我提过，别给他增加负担；(2) 不准给我爸打电话说有关的事，有事直接联系我或者我妈；(3) 尽量啥也别干，少添乱，如果谁做了任何事影响到我爸，我跟他没完。

当然，也得和他们说清楚，病能治；虽然治起来很不容易，但不是就等死了。

要说这些话，光是想想就烦躁、就疲惫。

学校的事：课程要等周六再选。助教的事，希望明早能看到邮件；如果收到，就尽快联系老师、说明情况，问清楚远程要做的事。等到周一，还要发邮件给Anton，再说一下推迟回去的事。

剩下的，虽然其他工作都可以推迟，但Sabre的面试，回去美国前最好能大致准备出来。

很累，早早睡吧。

January 3rd, Friday #

上午二姨夫忽然说，想带我去见延边医院的一个大夫；说对方和他刚认识，有个电脑相关的项目想问问。我内心猜测，二姨夫也是想借机问问老爸的病，就一口答应了。见面后听这位周文静大夫胡扯了一通，无非是个自动诊断的小软件。临走时二姨夫果然问了老爸的事。出乎意料，周大夫一下子说了不少不乐观的话，又提到之前的检查结果种种不规范，顺带表示国内骗人的太多。

一下子把我吓破了胆。

回姥姥家收拾东西、吃饭、买感冒药，然后跟小姨去高铁站。一路上脑子里都在想，必须得带老爸去上海看看，把情况都弄清楚。到车站后本想直接电话打给老爸；再三斟酌，总算没有冲动，而是先给张叔打电话沟通了一下，说明想法。高铁不停地过山洞，信号极差；因此到吉林打上车之后，给淳映发了微信，请她帮忙打听上海的好医院、好大夫。淳映先在微信上问了，又说五点钟打电话来。这边联系过淳映，那边又陆续收到了张叔发来的许多资料照片。一路焦躁烦闷，最后还走错了路，但总算顺利到了三叔家。

到达后，只见三叔二叔以及某位不认识的“大爷”在喝酒。三叔倒还可以，但“大爷”和二叔那些虚头巴脑的指指点点，已经把我的耐心逼到了极限。毫不留情地冷言了几句后，我就去到另一个房间呆着，整理资料、和淳映通电话。期间二叔来找我几次，似乎也觉察到不对；我只是不去理睬。

等到把资料按照时间顺序整理过，又按照淳映说的，去“好大夫”上提交了几份图文资讯，心里才渐渐平复过来。以手头资料看，诊断和治疗过程比想象中细致很多。“好大夫”上的回复，看上去也都乐观。

只是同时看到淳映在发朋友圈，又在“吃货群”群里打趣；才意识到自己的事对于她，只是生活中不起眼的一朵小浪花罢了。既然如此，过几天把话说到，之后找机会还上人情，也就无可计较了。

想到这里，心里难受。

晚上和三叔、三婶聊到很晚，精神总算慢慢放松下来，心里温暖了许多。

January 4th, Saturday #

早起吃饭就去上祖坟；之后宏伟、洪阳到家，聊了阵子天，然后出发去法特车站。期间找机会和三叔三婶说了老爸身体不好的事；也和宏伟说了，叫他稍微转告洪阳一下。至于具体病情，我没说，也反复叮嘱了他们别去问，免得给老爸压力。至于二叔和大姑那边，决定还是不说了，不放心他们。

坐在车上，重新想老爸的治疗过程、各种诊断，以及网上大夫的零星回复。想来想去，觉得张叔做事的确非常细致，和我家人的沟通也非常周全，让老爸的治疗基本没走弯路。就算是作为亲儿子的我，如果回来照顾老爸，也绝不可能做的比他更好。想到这里，觉得很安心，也很愧疚。于是联系张叔，说了这些想法，又说自己昨天太着急，说话没轻重，请他一定别介意。张叔说没关系，安慰我了几句。

回通化的大巴有些绕远；十一点半从法特出发，六点半才到家。老爸开车来接，我赶紧催他回车里，别吹了风。回到家里吃晚饭，和爸妈好好聊过天，算是开诚布公，心里也轻松了很多。以现在的情况看，下一步就是准备手术。当然，我回到美国后，依然要仔细了解情况，尤其是了解预后、防止复发的问题。

只是看到老爸在喝中药，有点担心；但听老爸说北京的这位中医可靠、吃了药明显恢复得好，也就无话了。

到家放松下来，才知自己疲惫，早早睡下。

真心祈祷一切顺利。

January 5th, Sunday #

从早到晚都和爸妈宅在家，久违地度过了幸福的一天。除了吃饭聊天，上午把新学期的课程选了一下，晚上则准备了介绍信等材料，给爸妈办签证用。在"好大夫"上的咨询：傅传刚主任说不能排除肺部转移；虽然是场面话，也让我立刻心惊胆战。好在许建民主任说肺部转移可能性不大，这才放下心来。

之前说为老爸的病，决心不再做“网约SM”的事；现在觉得还要再加一条，每天拉伸。如果只是“不去做什么”，总觉得提醒的力度不够强。每天要做的事，能更好地提醒自己，不要拖沓，不要托大，要认真对待。

January 6th, Monday #

不知为何，一天身体都很疲惫。早晚做了些杂事，中午则是和卢国文出去吃饭聊天；聊过天，心情好了些。晚上张叔来电话，和爸妈说了要再增加一次化疗，然后预约手术。老爸明显不想再做，但没办法，心疼。

和老妈聊天，大致确定五月带老爸出国，再细看病。在考虑暑假留校教课，这样可支配时间更多些。不过如果实习开始较晚，且检查结果较好，实习倒也不妨事。

晚上又开始发烧，38度整，愁。吃了点药，希望明早能好。

January 7th, Tuesday #

早上去打了吊瓶，身体好转很多；为了不再反复，算上今天，需要连打三天。大夫人很好，诊所就在门口，也很近，体验还不错。昨天在想，我这次得病，也许是给老爸挡了一次。这么一想，心里舒服很多。

定下了后天去延吉，打完针就出发。之前不想爸妈跟着折腾，但是拗不过，只好跟他们一起去了。老妈提议找个人帮忙开车；老爸觉得自己能开，我俩不放心。我提议我开去延吉，但没有国内驾照，有点不稳妥。

之前被分给了Serban当助教；但领队的助教一直不回信，课程情况也是一团乱。先静观其变好了。

January 8th, Wednesday #

又是一上午打吊瓶，这次是老爸陪。本来是回家照顾老爸，结果变成被老爸照顾，心情很复杂。

白天除了和家人聊天，没做什么有意义的事。晚上和爸妈商量手术方案；北京两个团队，北大医院的李子玉建议肝转移灶用切除，科学院肿瘤医院的毕建军说可以射频消融。老妈觉得切除更彻底，但老爸对切除非常抵触。老爸的方案要尽快定下来，这里也有张叔17号要去日本陪老婆看病的原因，老爸因此很烦躁。

我先去网上整理了一下情况，又去“好大夫”上咨询了一下之前接触过的三位医生。目前的情况看，切除最彻底，但创面大，需要医生技术很高；射频消融效果也很好，可以反复进行。如此听下来，我个人是倾向于消融的。明天约了一个付费的电话咨询，到时再细问下切除是怎么切，以及医生怎么建议。

之前 (1号) 联系了北京的新药实验，今天方才得到回复。为了不增加额外的复杂度，暂时先搁置。

晚上整理东西，然后拉伸。之前周日开始做拉伸；周一时因为感冒，做拉伸被老妈制止，只好等到周二中午补上。周二晚上拉伸过，算上今天，已经进行四天。希望之后能一直坚持。

January 9th, Thursday #

早起洗澡、整理行李，然后去打吊瓶；回家后就出发去延吉。老妈找了下属来帮忙开车，安心许多。五点半到达延吉，二姨、二姨夫安排吃饭，饭后回家聊天。话说开了，气氛热烈，大家的心情也就轻松了许多。

家人各自分头问，我也约了电话咨询；商量下来，基本确定是做射频消融，而不是切除。切除的话，多半要做锥形切进去，创面太大。大家又宽慰老爸老妈了一通，嘱咐说尽快安排好手术。之后私下和二姨夫聊微信，二姨夫反复叮嘱，要多打听预后防止复发的问题；说毕竟有转移，一定要小心。我心里记下。

助教和学费的事情，发了两次邮件，依然没有回应。决定回学校再处理这两件事，希望不算晚。

January 10th, Friday #

之前忘了记，但昨天早上打吊瓶时接到了陈鹏的电话，说家里进贼，丢了很多东西。大致问了下，确认我的房间基本没被翻动过，至于有没有少东西，还得回去再检查。

（想起家里还没动过的鲜花饼，心想雨婕果真是灾星；想到这又摆摆手，觉得自己不该这样想。）

早上陪老爸去隔壁市医院做体检，包括血常规、尿常规、肝功能、肾功能、血离子；检查很顺利，结果也暂无意外。除了跑前跑后之外，我也在网上约了一份体检解读；老爸看了，大概是觉得欣慰的。

下午和老爸去洗了个澡。除此之外，今天主要就是理快递、收拾行李等等。

January 11th, Saturday #

昨晚不小心说了句不大吉利的话，不知道老爸有没注意到；早起前做了个莫名其妙的梦，大致是洗衣机洗不干净，也令我有点介意。起床之后，发现落枕了，一阵心烦。感冒也没好利索，让我有些担忧。

但话是说破无毒，梦也是反着来的；至于落枕和感冒，希望是又帮老爸挡住了些灾祸吧。

想到这里，又想起昨晚忘记拉伸了，吓了一跳；赶紧一大早补上，顺便多做了一次。

上午理好东西，中午出发去机场。临行前拥抱了老爸，叫他加油。写这篇东西的时候，我已经在仁川机场。接下来要飞亚特兰大，又是十多个钟头；到达的时候，是美国时间的周六晚上。

今天是很长的一天，时区意义上。

January 12th, Sunday #

大睡一觉，然后早起整理东西、购物、做饭。下午去找朋友们爬山、聊天、吃饭；到家很困，早早睡下。

昨天坐飞机时，有一段非常颠簸。说起来，其实也没有很严重，乘务员和其他乘客也都没作反应。但不知为何，那时我却忽然非常害怕。有一瞬间，竟觉得自己也许没办法安全到达了。恐惧之后，又担心爸妈听到消息会怎么想，之后要怎么办。想到这里，不禁悲从中来。

人在极度恐惧时，真的是会祈祷的，因为到了那一刻，才最明白自己是多么的渺小和无力。那一刻我想了很多；关于人生的意义，关于宗教和信仰，关于未来自己该做什么：想要理解真理，想要做“布道者”。

感恩生命，努力地去生活。

January 13th, Monday #

回到学校第一天，有些流水账要记。上午去上了“教学培训”的课，老师很有趣，课程看上去也很有用。下午见了Anton，约定每周一下午见面；除了展望研究，也和他说了些家里的情况。白天剩下的时间都在忙活助教的事；跌跌撞撞，总算是开始干活了。顺便收到了Airbnb的面试，接下来要准备。

还有很多杂事，这周内要处理，这里不再多记。

接下来几周，除了钉住爬住研究、上课、助教三条线，还要准备面试。等面试大体结束，就要决定暑假是实习还是上课。再往后，打算联系下物界科技，看看有什么研究可做。

另一方面，也要尽快联系医院。想到下月1号还要去夏威夷，下面几周真的会非常忙碌了。

January 14th, Tuesday #

虽然周日阳光很好，但之后两天阴雨不停；到现在，不觉得时差有完全倒好。落枕依然没有完全痊愈，而感冒则在慢慢加重。下午上课时，觉得似乎又开始发烧；下课去医院，被告知说明天再来。开车回家，虽然四点半就出发，路上却非常拥挤；堵在路上，心情十分沮丧。

明天可能会病更重，却还要自己开车去看医生，更不知能不能见到、能不能得到有效治疗。想到这里，真是连哭都哭不出来。人不走运起来，糟心事真是接二连三，只能祈祷一切都会好。

好在正事和杂事都在有条不紊地进行，只是不知道之后两天还能否继续高效工作。

只能一件件地去处理遇到的事，只能祈祷一切都会好起来。

《孤独的改革家》

日前一时冲动，换了只新的苹果表。新的设备带来新的动力；这阵子被疫情荒废的健身计划，现在全都恢复了生机。为了趁热打铁，我重新整理了日程表，充满激情地开始推动新一轮的“自我改革”。

改革一旦开始，就不可避免地扩大化。既然有了新的健身计划，所有其他的计划也都要更新一遍。计划一更新，环境就不能拖后腿；无论是衣柜、壁橱，还是电脑里的书库、资料夹，全都得收拾整齐才行。凭着这股子劲头，不出一天，我就把自己折腾得精疲力竭，黯然叫停了这次轰轰烈烈的改革。

从这一次，乃至之前几十次的失败经验中，我深刻地认识到一个道理：运动式的改革和治理，只能产生一时的效果。要保住成果，还是得沉淀出新的习惯和制度。激情会过去，热点会消退；趁着它们还在，盯住一件小事，短平快地做出点实质改变。要是目标扩大，各种问题都跑出来，就只能告一段落了。

改革的关键，是重新分配资源。对一个人来说，最大的资源是时间。拿不出时间来灌溉，再好的花草也只是等死。新习惯只能比旧习惯省时省力，只能产出更高，最好能解放更多的时间，否则永远落实不下去。好比我一直想写读书笔记，从来也没写出什么。到现在，只摘抄，复制粘贴，主要是理出个知识框架。到现在，也有点规模了。

说到这里，庆幸对个人而言，“权力”还是高度集中的。再怎么分配资源，也顶多是戒断反应，难受一阵子。要是去革一群人的命，断人财路杀人父母，不知有多难。

虽说渐进好过搞运动，但热情总会过去。没了热情，连渐进也没有了，宁可缩着不动。这么说来，压力比动力有用。引入了外部的压力，内部才能团结一致。请老师找教练，治不了懒病。被生活按在地上摩擦，治百病。

总之近期试着联系下二姨，看看能不能和姥爷视频吧。在不添麻烦的前提下。

October 15th, Thursday #

周二和下午 Anton 视频后，开始弄 mc joint 这个章节剩下的一个引理。本以为很快就可以完成，结果连着燃烧了两天，到今天中午才勉强做出来。算了下时间，前后差不多花了十五个小时。

本想在明天出发去大烟山之前，把 ISyE 6383 的期末卷子批好的；现在看来，时间有点紧张。晚上雨婕还要来问题，陈鹏还叫我去买大烟山的饭菜。还是得能省则省，晚睡早起一点。

等周日回来，还是想把 nl joint 的东西稍微做一点出来，哪怕不是很完整。还有 mutli-attribute 的附录，也要尽快弄好。不知道周二见 Anton 之前能准备出多少东西来；但下周末之前，要尽量整理好这几块。

October 16th, Friday #

昨天证明晚引理已经是下午；收拾了下行李，就已经是晚饭时间了。为了给雨婕答疑，就没跟陈鹏去买菜。结果并没来得及在出发之前，去批什么期末卷子；旅途中大概也不会批。看来只好等回来再说了。

昨晚小白忽然有事，不能一起去大雾山了；少婷“及时表演”，又搞出了一次风波。早上起来，看到一个叫Tony的人在群里质问，说小白不去也该付住宿费。我怒火中烧，直接回怼，说了几句重话，刀刀去捅少婷。虽然稍后少婷和Tony的女朋友Heidi出来解释了几句，但总之整一天的旅行都有些尴尬。

早上给二姨打了个视频电话，看了看姥爷的情况。姥爷因为在吃止痛药，有些神智不清。我这时觉得，姥爷大概挺不过这一次了，又担心姥姥能不能吃得消。晚上见到大家，却听说同舟的姥姥下午去世了，一时心情复杂。晚饭时同舟拉着大家喝酒；先灌倒了陈鹏，又灌醉了自己，喝得直喊直哭。我和丁烈骁照顾两个醉鬼到深夜。

充满drama的旅行。

October 17th, Saturday #

因为昨晚的折腾，大家很晚才起床；出门时已经是十一点。开车去Cherokee转了下后，就直接去Clingmans Dome。由于之前少婷的各种折腾，我对组织工作毫无兴致，因此出了很多纰漏。不仅午饭没吃上，还“弄丢了”Tony他们的那辆车。但到了晚上，大家的气氛反倒变好了些，吃饭也到了一张桌子上。

之后终于成功生起了篝火，大家聊天到半夜。

October 18th, Sunday #

十点钟退房离开，走西线出大烟山。Tony等四个人说要早点回去，就直接离队了。比起昨天，今天的旅行倒是成功的多，景色也更惊艳些。只是午饭依然没地方可吃，手机也还是毫无信号。

本来觉得之前两天给同车的大家留了些好印象。但到今天晚些，总觉得柳月对我变得十分冷淡，梓莹也话少了很多。可能是我今天叨逼叨太多，说错了什么话；也可能大家只是太累了。又也许，我只是太敏感想太多。

烈骁昨天跟少婷聊了一阵子，今晚来找我们说，希望大家能和好。我和陈鹏吐槽了少婷一顿，说了很多让烈骁意外的事；他因此决定不去再管少婷。对我而言，也是跟少婷保持些距离的好。

October 19th, Monday #

前面耽误了几天，但总算批完了 6383 的期末卷子；这学期的助教工作就告一段落了。

October 20th, Tuesday #

本该开始努力推进研究，但完全不在学习的状态；加上Anton有事，索性水了一天。

Octoberc 21th, Wednesday #

又是一整天不想工作。但觉得总该干点什么，于是补齐了日记、日程表和计划，又开始整理关系网络。半年以来自己都在沉迷“国师学”，但这些“重工业”目前作用有限。还是得先在人际能力、人际关系上多下功夫。

October 22th, Thursday #

中午和高璇聊了一会。她在考虑申请博士，因此找我了解下流程。既然她大致想申金融精算方向，我打算介绍肖巧雨和黄梓莹给她认识。前者可以跟她搭伴，也可以介绍UGA；后者可以介绍GSU。我不妨也借此机会，去联系下各种朋友，了解GT相关专业的情况。只是一时还想不到联系谁。

昨天刚说要少看点“国师学”，多练练人际能力；今天上天就送来了机会。至于结果如何，就得看自己的操作了。今天或明天，打算先联系下巧宇，让她们交换微信；然后看在我去西雅图前，能否三人一起见面聊聊。

October 23th, Friday #

昨天听了雨婕的话，终于下定决心取消西雅图的行程。一来疫情风险太高，二来姥爷住院恐怕随时有变，三来研究压力确实不小，很多活没干完。下午跟陈鹏商量过后，折腾了一通，总算退掉了所有的机票和酒店。除了请陈鹏吃串以外，没有造成什么经济损失。

跟张灵、林婷婷说行程取消，两位自然有点失望。晚上和林婷婷视频了一下，算是 catch up。

October 24th, Saturday #

今天又花了时些间，思考了一下自己“两重量轻两服务”的技能格局。“两重”，一边是数学和计算机，一边是社会、政治、经济和管理；前者带动“量化交易”，后者带动“人脉圈子”。“量化交易”、“人脉圈子”这“两轻”也是相辅相成的关系：交易赚钱可以用来扩展人脉，从人脉获得信息可以辅助交易。以上这些，都要建立在“两服务”的基础上，即身体和精神的修炼，以及相关“衣食住行”等习惯的养成。

博士五年，目标是基本完成这“两重量轻两服务”的建设。如果之后能去好大学教书，自然最好，可以借助学校和学生更进一步；如果不行，凭这些也可以养活自己，再作下一步的打算。

目前来说，这几块内容我做的都不好；不过最近想着重加强一下人脉圈子的建设。如果只是闭门造车，实在是难以保持良好的精神状态。“为有源头活水来”，人还是要做社交动物才行。

October 25th, Sunday #

上午干了些活，下午去跟凯朝打网球，晚上约了杨柳月和贝尔来吃火锅、玩桌游。

做研究依旧不在状态。

October 26th, Monday #

研究了一下陈鹏、Rosy生日的时候该送他们什么。

之前总觉得，礼物就是要实用。为此，要多了解对方的需求，甚至提前“种草”、创造需求。但现在越来越觉得，礼物就是礼物，不能太实用。重要的是“定制”和“体验”，是独特的回忆，存在感要强。

比如对陈鹏，如果预算足够，可以送保时捷试驾、直升机试飞等等；对Rosy，可以约定制的摄影或者肖像画。考虑到时间和金钱都有限，打算去给陈鹏订一份和宠物一起的肖像画，用照片的那种。

October 27th, Tuesday #

之前花一周理好了 multi-attribute 的稿子，又花两周做 joint optimization 的证明。MC的证明用一周做好了，但NL一直做不出 (也做得不太用心)。今天和Anton聊，说打算再尝试一周，不行的话就先放弃。Anton说同意，毕竟这个东西一直没人做，肯定不是大家忽略了，而是确实不好做。

如果下周确实还是做不出来，最好还是暂时放弃，转而先写出一篇 MC joint optimization 的稿子来。之后一边整理和提交写好了的三个稿子，一边再想剩下的成果怎么发，顺便多读些文献。

October 28th, Wednesday #

之前突发奇想买了激光脱毛仪，没想到今天就到了，于是用了一番，等之后看看是否有效。其实后来才听丁烈骁说，他的亚马逊羊毛群里之后会送这个。不过也懒得退了，就少占点便宜吧。

October 29th, Thursday #

在Gallego那篇GAM的论文里发现了许多有意思的东西，之前竟然忽略了。还是要认真读论文。

关于 NL joint optimization 的证明，虽然猛攻了两天，但依然岿然不动。决定放弃。

October 31st, Saturday #

之前一直觉得，想读些人际关系的书，却找不到质量好的资料。今天在B站上看到一个教人追女的视频，看上去意外地很靠谱；随后又发现了一个日本综艺《有点心机又如何》，感觉十分有趣。接下来不妨研究一番。

November #

November 1st, Sunday #

今天过生日，叫了杨柳月、高璇、朱贝尔，以及丁烈骁、孙同舟来家里。晚饭做了海鲜锅和沙拉碗，加上陈鹏买的千层蛋糕，七个人大吃了一顿。饭前饭后都在打一夜狼，玩得很痛快。晚上和刚从大雾山回来的雨婕见了一面，收到很多礼物，感慨她的用心。

也收到了谷甘琪、林婷婷、马思爽、白伟成，以及时国慧和黄少婷的生日祝福。看了看这个配置，感慨自己交友不用心；以至年纪愈大，记得自己生日的朋友愈少。反过来，自己也少有记得别人生日的时候。

还是要多用心。

November 2nd, Monday #

今天久违地状态很好，把屋子和厨房都打扫了一遍，又处理了不少杂事。大概昨天生日过得很充实，改善了我的心理状态；且杨柳月终于把狗接走了，家里清静了不少。

November 3rd, Tuesday #

大选日，晚上打过网球后就一直在看直播。出乎意料地，川普支持率很高，大有要赢的态势。不过到了最后，形势依然胶着。考虑到邮寄选票，剩下的六个摇摆州会逐渐“蓝移”；搞不好拜登以微弱的优势胜选。

如果是这样，那这次大选就真的成了“梦幻结局”了；因为川普大概率会不接受结果，双方闹到最高法院。川普的支持者体验过“得而复失”后，很大概率会上街。如果川普本身煽风点火，后果想必非常可怕。

最近得非常注意安全了。

November 4th, Wednesday #

选情胶着，社会冲突已经可以预见，但美股却在大涨，让我十分困惑。事实当然不会出错，出错的只能是我的分析；但我实在不知道如何解释现在的现象。难道美股已经成为了避险港口？

November 5th, Thurday #

越来越觉得，诸如“顾子明的政事堂”、“吃瓜群众岱岱”、“香山劳大叶主任”之类的时政公众号，其实价值不大。一方面信息太碎片化、逻辑不成框架，一方面又在卖弄本事、贩卖焦虑。

但即使看了那些政治经济学大师们的视频，建立了思维框架，还是难以落到实处去。如果没有具体的资源和信息渠道，再好的框架也难为无米之炊。说到最后，还是要锻炼微操的能力。

至于投资赚钱，还是靠数学要稍微靠谱些。

November 6th, Friday #

晚上临时找了一堆人来玩桌游，还蛮过瘾的。不知道能不能构造一个稳定的桌游小圈子。

November 7th, Saturday #

这两天跟甘琪和李朔聊了不少，今天上午和晓晨聊了一通，下午又找雨婕和国慧他们爬山。虽然跟很多人在聊，但还是缺乏章法，没有有意识地去聊，也没有保证连贯的推进。还需要联系。

November 8th, Sunday #

本来觉得工作不算紧张，结果今天一整天都在放羊，现在真的紧张了…

November 9th, Monday #

看到了一本叫《原子习惯》的书，觉得应该买来看看，然后设计一下自己的“习惯系统”。虽然意志力很重要，但设计科学和巧妙的习惯体系也很重要。最近的想法，是每次性行为 (包括自慰) 之前，都必须洗冷水澡。

November 10th, Tuesday #

目前正在努力完成第三篇论文的稿子。接下来要思考的问题，一是怎么能让Anton快点把论文改了，二是接下来该做什么。目前很想去做交通相关的问题，但担心研究和选择模型完全无关，多花时间且和之前的作品不成体系。

Tesla被录入SPY，再次大涨；难以承受，最后还是选择割肉退场。之前在股市小进小出很多，大抵不亏不赚；这一次直接亏了一万刀，实在惨痛无比。痛定思痛，总结经验教训。

January 19th, Friday #

胡珺琪从亚特兰大面试归来，找我和煜玮、大为吃饭。商量了半天，最后约在一个叫Quartino的意大利馆子。一起来的还有胡珺琪的一个朋友，说是追韩团认识的，在UIUC（芝加哥）读Accounting的博士。虽然没问叫什么，但她也住在River North。不知道之后有没有机会认识。

Quartino味道一般，只有披萨和点心还行。倒是借着约饭的机会，体会了一下零下十几度在街上走路的感觉。只能说身上感觉还好，但帽子手套一定要带，最好再带上面罩 …

January 20th, Saturday #

又和珺琪、煜炜、大为约了午饭，顺便吃了点心。昨天一起的妹子今天没来，有点可惜。不过饭后理了发（三十多块，好贵），还打包了一份烧鸭回去吃，也算收获不小（笑）。

晚上和爸妈视频，聊了三个小时；之前东奔西走，感觉冷落他们太久了。

January 21st, Sunday #

下午和老巴视频，意外地又被他训了一顿，说我还是总是自己瞎搞，不听他指挥，也不好好按照现有（Ebru）的东西做。我本以为自己已经在注意了。看来，我这个总是“lose focus”的习惯已经太根深蒂固了。得治。

另外也被他吐槽，说我总是看不到邮件，说既然给他发了邮件等他回，自己这边就要准备好才行。于是专门设置了手机，保证他和Anton的消息即使在静音状态下也有提示。设完又想到，这算不算职场PUA？可就算是，我也得“顺毛捋”才行；就像茗晰说的，多跟煜炜学学，好好向上管理（笑）。

January 22nd, Monday #

折腾了一整天，连蒙带骗，总算把给王凯和曲老师的课程slides弄完了。说真的，实在不太喜欢做这种东西。

这两天收拾了一下Raspberry Pi，在上面装了WordPress。想试试能不能弄个用户注册和收费的功能，之后有用。

January 23rd, Tuesday #

快两周没去学校了；今天趁着有seminar和免费饭，过去呆了一阵子。说真的，中午过去还能停车，实在是运气。

定了三周以后去亚特兰大的机票，顺便和张薇聊了聊，帮她参谋AI作业怎么写。Oculus也到了。玩了一会，感觉不错。

January 24th, Wednesday #

晚上和茗晰打游戏，一不小心弄到很晚。想着马上睡也未必睡得着，就一边听书一边收拾屋子。正好听了一本关于睡眠的书，里面提到光照对睡眠的影响；这才想起来台灯和落地灯还没买。于是去Amazon连续下单，准备改善一下屋里的光照环境。

January 25th, Thursday #

上午去学校，但完全找不到车位；转了一大圈，只好默默回家。于是暗下决心，之后要早起。回来后听茗晰说，GSU商院的teaching faculty岗位给别人了，胡珺琪应该拿不到了。可惜。

今晚终于又觉得心静了些，即使呆在屋里什么都不做，也觉得蛮舒适。过去三个月，总觉内心总是很躁动，很难平静下来。最初以为是奔波太多，独处太少；但在芝加哥独居了几周，也没觉得有什么改善。

17号刚回来时去Trader Joes买了几支百合花。当时还觉得很少，没想到越开越多，很快就“爆炸”开来。到今天，状态依然不错，只是一些花瓣开始干枯、凋零。稍微收拾、整理了下，立刻又显得娇艳无比。

拍了照发给老爸老妈，得到评价：买的挺值。

January 26th, Friday #

周一的时候，Anton发邮件来，说要用pickup-time那篇文章讲个seminar，问我OneDrive里的内容是不是最新的。刚刚发现，他之后几天都有在OneDrive里改文章，刚刚把摘要和简介都改完了，不知道会不会继续往后改。只可惜他要讲的seminar就在下周五，恐怕他不能改很多内容了。但改就比不改好。

January 27th, Saturday #

昨晚和茗晰吵了一架。大体上是她想要娇嗔一下，表达一些情绪和不满，但却触发了我对“被指责”的反感。我于是愤怒回怼，而这又触发了她对于“不被允许表达情绪”的反感。于是越吵越凶。这个情况已经发生过几次了。今天和她好好聊了一下，希望之后能逐渐不再陷入这个“习惯性吵架”的坑。

晚上突发奇想，用Heygen的AI Translation做了个“内塔尼亚胡用中文回应海牙法庭”的视频。本来觉得是个很好的点子，但发到B站上后并没什么人看，小红书更是直接限流。之后搜了下，发现B站上不乏类似的视频，很多人的点子比我更好（比如让胡赛武装的叶海亚用中文讲毛选），但观看量也不算多。于是感慨，有点子就要尽快去尝试，试过才知道大多点子其实行不通。如果不去尝试，总在那反复想，还真以为自己多厉害（苦笑）。

January 28th, Sunday #

昨天把做的“内塔尼亚胡”的视频顺便发了微信视频号。今天早上起来后，意外发现视频火了，有好几万的观看。更意外的是，视频下面大多数人都在支持以色列。我一开始以为是进入了另一个“信息茧房”，而茗晰则觉得我是被以色列水军盯上了。现在想想，大概两者都有。茗晰提醒我注意政治影响。确实要注意。

当然，这次意外还是给我带来了几百粉丝。我本来还在努力思考接下来要发什么，是迎合粉丝，还是发个反以色列的视频为自己“正名”。后来觉得，两者都不必，还是冷却几天，静观其变的好。等理解了这些粉丝的构成、偏好，并且对微信视频号的经营有了一些认识和规划，再决定下一步的好。

于是又有个感慨：之前我的很多点子，其实都是“拿着产品找客户”。对于客户（比如微信上的“中老年群体”）的真实喜好，我简直一无所知。丁老板说得对：谁是你的客户？这个是做生意的首要问题。

January 29th, Monday #

早起成功（终于），但还是不够早，停车位很难找。

晚上又尝试发了一条AI视频，是奥巴马用中文说龙年快乐。发之前觉得自己这个点子简直是天才，发之后却反响平平。看来果然还是太自信了（笑），对营销和用户喜好都太无知。

January 30th, Tuesday #

昨天八点出门，八点半左右开到学校附近，发现特别堵，完全找不到车位。转悠半天，停在学校里面、系楼北边的一个地方。今天整体上比昨天早了十分钟，但更堵了，最后听得也更远。看来八点到八点半是家长们集中送孩子的时间，还是要避开。（说起来今天也是不太顺。早上一出门就发现高速匝道封了，绕了一大圈到学校却没停车位；尿急加肚子疼地到了办公室，却发现保洁女士正在慢条斯理地打扫洗手间，只好转战系楼。）

January 31st, Wednesday #

早上又是花了很多时间找停车位。算了算，从家门口到工位，差不多花了五十分钟。如果总是这个时间，那真不如直接去做校巴，路上还能听听书、写写日记啥的。

February #

February 17th, Saturday #

工作一直忙到了昨晚，到今天上午才开始收拾行李。忽然又收到了Baris的邮件，问我中午有没有空meet，让我大呼倒霉，跟茗晰吐槽说怎么每次都赶着我要坐飞机的时候meet。没办法，还是跟Baris之说我要飞亚特兰大，然后在机场meet了一下。

他也已经在航站楼（不过不是一个机场），只是简单聊了几句，给我安排了下工作，然后就挂断了。感觉他气色很差，似乎不是因为我，而是本身状态不佳。好奇这是因为啥。

下午飞到亚特兰大，和茗晰团聚、吃饭、打游戏。

February 18th, Sunday #

上午约了李延和海容一起吃早茶，八卦了些学术圈的事。感觉李延找工作并不顺利，到现在也没什么很好的offer。我还是试探着再劝他联系Baris，这次似乎劝动了。说实话，我现在才渐渐发现李延真的很强，最近这篇论文已经是高睿的水平了；所以也就能理解，让他跟我一样去跟Baris做博后，大概会让他不太甘心。不过形势比人强。不管怎样，希望他能有个好出路。

茗晰这两天非常忙，今天又临时被安排了工作，于是下午回来干活；我也就正好把Vast.ai上的服务器设置了一下，以便明后天写代码。

可能是因为刚出生理期，且压力大、状态差，茗晰莫名地因为我干活时没理她而开生气，并且因为晚上吃饭的地方太远而越来越气。我这次倒是没怎么跟着生气，也没和她吵架，只是默默安慰。

晚上约了莱克西、涵一、雷雷、丁老板吃饭。他们算是刚知道我和茗晰在谈，我俩因此被打趣了很多。吃饱聊好，茗晰的心情也好了很多。

然后晚上回家，被我皮带绑手，“教训”了一顿（笑。

February 19th, Monday #

白天去学校干活，晚上回茗晰家。回家时发现梦雨和佳鹭在客厅，茗晰于是把我塞进屋里，然后出去找她们聊天了。我被晾在屋里，心情难受，于是跟茗晰吐槽了一通。昨天她刚生了气，没想到今天我又生了气。

好在她也认得快，俩人也没怎么吵架。

February 20th, Tuesday #

和昨天一样，白天干活，晚上回家。因为昨天的事，今天茗晰约了梦雨和佳鹭一起吃饭。刚出门时我和她们有些尴尬，但总算最后聊得还不错。希望给她们留了个好印象。

顺带一提，“马路边边”确实好吃，但浑身沾的火锅味，实在太可怕了。

February 21st, Wednesday #

又忙活了一天的编程，但毫无进展；本以为要被Baris臭骂，但他似乎觉得这是预料之中，之说让我想想下一步怎么办，写个计划给他，顺便问问实验报告该怎么写就行。

之前跟Baris说我想在亚特兰大留到周六，这样周五可以听听Jose的seminar；他当时幽幽地回了一句“这跟我们的研究有啥关系”，把我吓了一跳，也是以为要挨骂了。不过今天视频，感觉他对于我到处跑没那么在意，倒真的只是觉得不要distract。我于是跟茗晰吐槽说，早知道就直接说是来陪女朋友了。

February 22nd, Thursday #

上午收拾了东西，带着茗晰搬去陈鹏家，跟张薇聊了很多，顺便吃了午饭。聊到陈师傅的签证经历，以及张薇在小红书上看到网友的各种签证经历，我和茗晰都感觉压力很大。想回国签证又怕出状况，不回去又觉得不自由。愁。

中午吃饭碰到了佘钊伟的老婆恒一，听她吐槽了很多新加坡的社会现象。我之前一直觉得新加坡好，但对新加坡政府的父权式管理、朝令夕改，以及新加坡社会的活力低下毫无概念。看来还是要多了解。

下午应付了一下工作，给Baris发了邮件，然后带茗晰去Costco看轮胎。回到家后不知为何，两个人都觉得很疲劳，就默默地宅到了睡觉。本来这次回亚特兰大，还想着去看Hamilton巡演、看《热辣滚烫》电影、攀岩啥的，结果又忙又懒，这些事一个都没做成。想想还稍微有点忧伤。

February 23rd, Friday #

终于去听了Jose Blanchant的seminar，一进屋就听到Jose说，自己的这篇文章是受到了Alex (Sharpiro)的启发才写的；讲talk的过程中，也认真地谈了Alex和李延的贡献。我不禁感慨，学术江湖果然就是人情世故。

之前觉得只是Jose和Alex在竞争这篇文章；听Jose讲完，才发现是很多人都在各自做同一个问题的不同侧面，而Jose和Alex只是其中两个做得比较好的而已。之前Jose和高睿争文章，情况大概也相似，只是高睿被Anton拖得更久而已。这种“水到渠成”式的研究，虽然也需要很高的水平，大概远不是那种开宗立派的程度。不过即使这样，司念、李延和高睿也真的很厉害了。

中午和Myungeun吃了个饭，聊了很多，不过基本是我在叨逼叨。事后想想，也记不起都说了啥。惭愧。

下午又见了下Anton，被他布置了两个任务：一是增加一个新的section，谈time varying parameters；二是考虑下怎么把numerical results搞得好一点。怎么说呢，有点哭笑不得。

晚上又带茗晰去找朋友们吃了饭，还是莱克西、涵一、丁老板。区别是这次雷雷有事没来，而张薇来了，还带了朋友MJ。之前在视频里见过MJ，这次终于见到了真人。饭后大家又去陈鹏家撸狗打游戏，一直到很晚。

火柴人大乱斗确实好玩。

February 24th, Saturday #

很晚才起床，收拾好行李，然后和茗晰去陈鹏家旁边的小摊子吃三明治。摊主黑大哥依然那么轻松自在，感染得我们也跟着心情明亮起来。和茗晰吃了午饭，然后开车去机场；中间聊了很多对未来的想法，有迷茫也有憧憬。

上飞机时收到了大为的微信，问需不需要接机，顺便和他的室友白星宇个吃饭认识一下。问了下大为，得知白老师准备接港理工的offer，下半年要也去香港和司老师团聚了。查了查白老师和大为的论文，有点受打击。

February 25th, Sunday #

什么都不想做，无所事事一整天。晚上跟家里视频了一下，说工作不好找，签证也不好搞。愁。

February 26th, Monday #

工作效率不错，一整体干了很多活。感觉与其每天雨露均沾地什么都干点，不如集中精力一件事一件事地做，每次都要搞定一个节点，日拱一卒。

February 27th, Tuesday #

之前一直觉得，如果九点左右来学校，旁边小学送孩子的家长都走了，基本就有停车位了。昨天这样试了下，也确实感觉很好停车。可今天再来，所有能想到的地方全都停满了；转悠了半小时，都准备放弃回家了，才终于找到一个停车位。看来想要稳定地有地方停车，唯一的办法就是八点之前到。

March #

March 7th, Thursday #

工作终于有了进展。周一修复了程序里的几个bug，周二发现reference policy随机性必须足够大才行（而增加Y的dB没啥用而且会改变问题性质），昨天终于开始得到合理的结果，而今天发现我之前“统一网络”的方法可以大幅加快训练同时提升效果。

接下来打算把所有发现整理到一份文档里，生成PPT，先给煜炜和大为看看提提意见，然后给Baris看。如果没啥问题，就把内容加到论文稿里去。

March 21th, Thursday #

之前Anton一直催，今天终于在Booth讲了pick-up time这篇论文。这次seminar是三个人一起的，一个人只有20分钟，所以也没能讲得很细；但总算觉得发挥不错。Amy Ward也在，但只是开头提了个问题；后面seminar刚结束，她就起身走了，也不知道她感觉如何。事后给Anton发了个邮件。

跟心怡聊天，说打算如果接下来两个月能一直骑车通勤，那接下来就买个好点的车，小布之类的。

新同事许耀升来了，跟她和煜玮聊了蛮久。

June 11th, Tuesday #

公寓的管理员发邮件，要约时间给房门刷漆，问大家什么时候在家。我说周五下午可以。管理员问具体什么时候？回信还抄送了房东Ashely。我这才察觉自己的不周到：租了别人的房子，和管理员联系刷漆时间，竟然没有抄送房东。

又想起，之前茗晰来住，我也没和Ashely说；结果被管理员发现告诉Ashely，弄得她不太愉快。这次房门刷漆，不知Ashely会不会借故突然袭击，亲自跑来公寓看看。之前觉得茗晰在家就够了，现在觉得还是自己也回来比较好。于是问管理员，刷漆的时间能不能约得晚一点？

July 29th, Monday #

白天折腾了一下实验，在很小的问题上测试了不同的learning rate，发现之前看到的oscillating gradient，其实是模型还远远没训练好的表现。整理了一下实验结果，发给了Baris，打算明后天再详细看看。

晚上约了丁老板、雷雷、莱克西、陈鹏、张薇吃饭。之前周末就约了一次，但大家都很忙，约不上时间；这次约上了，却感觉这饭吃得也有点淡淡的，有点不知道该聊啥，没多久就结束了。

丁老板最近很忙，大概老婆也在怀孕，没时间攀岩；雷雷自从有了女朋友，除了偶尔吃饭，也很难约到。只有莱克西和涵一还能约来桌游，但涵一加班太多，这次也没来成。时间渐渐过去，总觉得有些关系没办法像从前那么密切了。

心情难过，不知如何说。

August #

August 4th, Sunday #

又是很晚才起，和茗晰出门吃饭，回来后收拾了下东西，就接上了佳鹭一起去机场。飞机延误了一个多小时；到落地时，乘客们气性似乎不小。机场也是人满为患，折腾了半天才打到车。

司机师傅是个天津人，一路都在和我聊天；以至于到家下车时，我都有点恍惚，以为自己是在国内。意识到自己还是孤身在异乡漂泊，心里有点难过。

August 18th, Sunday #

晚上和茗晰去Trader Joe’s买菜，刚要过一路口，后面冲上来七八个骑摩托的警察，把路口封上了。然后一个车队，十几辆车，开始从我俩面前过，前面几辆车里坐的人穿的是西装，后面车里都是穿马甲戴头盔的军警。我俩在那站了几分钟，才能过去买菜。

回来看新闻才知道，应该是哈里斯的车队，来参加民主党大会的。估计最近的时候，哈里斯离我俩就几米。

August 31th, Saturday #

昨天张灵说，他和段总学了个新习惯：每天早上做108个瑜伽“拜日式”。今早起床下楼，发现张灵果然正在“拜日”。我说这个不错，既简单又直接；本来我每天就该做拉伸，正好可以学来试试。随后付老师陪张灵去理发，又带了咖啡回来。穿戴整齐后，三人收拾出门，去法院和朋友们汇合。

主持付老师和张灵领证的法官，是个挺幽默的话痨犹太老头。之前Ruby和她的乌克兰老公领证，就是这个法官主持；当时张灵也在场，觉得这人不错；这次自己的仪式，就决定也找他。领证仪式非常温馨；张灵自己写了给付老师的结婚誓言，说结婚是对于“见证彼此的生命”的承诺；付老师则读了一首莎士比亚的十四行诗。除了付老师忘带驾照的“有惊无险”，是个挺完美的仪式。

亲友团里有“高地圈子”的段总和郭宇婷、张灵在Bellvue的邻居刘畅、张灵叫来的密友我、付老师叫来的密友安喆，以及张灵的老板刘欣和印度同事Kevin。这其中，刘欣老师和安喆是证婚人。至于熊和大姐，由于离婚的缘故，张灵决定两边都不叫。

典礼结束，大家在法院门口拍了些照片。之后，刘欣老师去机场接老公和孩子，Kevin也离开，剩下的人则去湘菜馆子吃午饭。我最初总觉得，刘欣老师似乎不太想理我；但仪式之后，她主动过来打了招呼，我就赶紧识抬举地自报家门、加微信。虽然还想多聊，一则刘老师急着走，二则我实在不擅长一对一聊天。于是计划等到INFORMS的时候，叫上张灵再约她。

晚上大家又去段总家吃晚饭、吃蛋糕。人大致还是这些，但是是多了熊和段总的室友心怡，少了郭雨婷。张灵解释说，郭雨婷不喜欢熊，觉得他是人渣。段总的无花果蛋糕

昨天、今天见了两次面后，我和熊之间又变得自在了些。稍晚些，段总也来了。大家打着游戏、聊着天，气氛像是和从前一样。只是时间紧。吃过午饭，张灵和付老师就匆匆送我去机场。

中午骑车来学校，看到大为和耀升在一起吃饭，就过去一起聊了会天。听大为提到，巴老师每年感恩节、圣诞节这两周都会雷打不动地陪家人。我忽然想到，如果我在INFORMS之后，也就是十月底去加拿大签证，然后十一月中下旬回来，就正好撞上了holiday season。在巴老师的角度看来，我就是两个多月没怎么正经干活。

October 7th, Monday #

Ebru，刘欣

October 8th, Tuesday #

张灵昨天帮我在Amazon的一个大群里发了消息，问有没有我能申请的工作。今天有些人来要简历，于是我花了一上午改好了简历，发给张灵让他帮忙pass around。

中午送茗晰去机场。

晚些张灵发了些内部资料来，说Amazon会有postdoc机会，是两年期，拿J1签证的。有个点是，开始时间要在拿到PhD的两年以内。对我来说，就是明年12月初之前。张灵的意思是，别只看full time，也要考虑postdoc，先一脚跨进来再说。进来了以后什么都好说，活生生的人总比一张纸强。

October 9th, Wednesday #

茗晰今天和Wei聊到J1签证waiver的事，Wei建议我先别急着搞waiver，因为基本上只能waive一张表，但我可能还需要extend我的J1项目，会有第二张表。他说，中方只会给出一次不持异议信；说的狠一点，就是就是一辈子只让waive一次。而美国这边，他会觉得你又收到了一份新的表格，会不会重新受到两年服务期的限制。如果受到了的话，可能需要中方给再给出一份不持异议信，但中方一般不给再出了。

我试着查了一下，没办法准确得知Wei说的是否准确，中方是不是只会给出一次不持异议信，但这种风险确实存在，是不少J1学者约定俗成的认知。而且Wei说，联系中方人员也会很难。除此之外，如果waiver被批准，一般就认为J1项目已经结束了；美国政府一般不允许项目继续extend，学校部门也会比较reluctant。

茗晰和Wei觉得，这个waiver一定要搞，但必须是项目即将结束前的最后一张contract拿到就开始搞，这样是时间最宽余，policy上也最不会出问题的。我于是把不持异议信的申请撤销了。准确地说，系统里没有撤销这个选项，但是如果修改，就是变回草稿状态，系统还特意说，只有再确认提交才回受理。

又查了一下七月份跟煜炜的聊天记录。她也说，豁免可能只能有一次，甚至J可能也只能有一个（不确定，可以问问胡珺琪？如果是真的，那Amazon的J1机会可能搞不定）。而且美方和中方都会要全部的DS2019；如果没工作，申请豁免就要三思。否则豁免之后，再延期，就不知道是不是一个case了。不如等有苗头了再开始。

我打算最近跟Stephen联系一下，先把Booth这边延期搞了；一来安全，二来对下次签证时长也有好处。等Booth这边的项目extend好，再看看接下来会不会去Amazon或者其他地方做另一个J1项目。煜炜的waiver用了三个月搞定，我初步打算预留六个月，以保安全。

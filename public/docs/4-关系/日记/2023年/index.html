    January #

    January 1st, Sunday #

    昨晚休息得很好，今天精神状态不错。新年第一天，白天大多数时间在看视频学东西，以及陪家里人聊天；晚上则分别和[[卢国文]]、[[张馨月]]视频聊天，聊到蛮晚。马上要动身去美国；有乡愁，有失落，也有兴奋和新的期待。

    老妈说，今年最要紧的任务，是博士毕业和入职清华；如果有精力和机会，顺便找个好对象。我很同意。

    除此之外，我还有两件想做的事：一是继续修行“正念”；二是多学RL和AI的知识、多练软件开发的能力，并借此赚些钱。关于前者，我近期找到了一种“放松地吸收”的感觉，让自己学东西和做事时不再急着输出和表达，而是不断地平静地汲取，日复一日地提高自己，并且享受这个过程。接下来一段时间，我想要巩固这种状态，让自己能高效做好“主业”，同时把“副业”和“学东西”变成自己的放松。而我要玩的“副业”和要学的东西，正是RL、AI、软件开发等等。

    January 2nd, Monday #

    虽然应该做点“正事”，比如改论文或者给曲老师写课程PPT，但感觉今天并没有这种心情。白天学了点OpenAI API的prompt engineering知识；晚饭后去买了个大托运箱，然后收拾了下行李。

    关于找对象，虽然之前一直“心存幻想”，也努力地在和[[王心怡]]、[[尤培钰]]聊天，但如果接下来去美国，恐怕回来前很难跟谁有进展。李朔那边也是同理。最好的做法，大概就是顺其自然，在国外时不必急切地联系；如果回来时还有机会，可以再尝试续一续缘分。倒是不知出国以后能否遇到新的机会：比如认识些陈鹏女朋友的同学，或是找熊和张灵帮帮忙。

    问题的关键，还是老妈说的：别老瞻前顾后，要积极主动点。

    January 3th, Tuesday #

    明天就要去北京，然后周日就要飞美国。想到这些，总觉得有点心浮气躁。上午稍微给曲老师干了些活，给V2V那篇文章做最后的proof of edits；在那之后，就基本只是做做杂事、刷刷视频。

    最近在思考AI的问题，发现与RL和区块链相比，用AI赚钱似乎更“现实”一些。这里面的第一层，是用ChatGPT等服务，去快速大量地完成一些简单低端的“兼职外包工作”；关于这一层，我打算做些尝试，比如去Boss直聘上找兼职来做。第二层，则是基于成熟的API做开发，比如OpenAI的语言模型API；关于这一层，我要用一些时间来建立写APP或微信小程序的能力，也已经有了些点子。第三层，则是基于现有的模型做进一步的训练；关于这一层，我暂时还不打算接触。

    前天和国文、馨月都聊到了这个话题，之后也可能会跟馨月尝试合作。但在那之前，还是不要过多地去说、去空想，而是认真、平静又踏实地一步步把事情做起来。不要整天幻想结果而耗费心力，只需把这件事当作业余时的放松，慢慢享受过程就好。

    January 4th, Wednesday #

    可能是身体还没完全从新冠中恢复，也可能是几个月来疏于锻炼，今天早起飞到北京后累的够呛，大半个下午都在睡觉休息。不过还是去取了之前修的鞋，开通了数字人民币的账户，也大致上理了下行李。顺便，还跑去吃了欧记的麻辣烫，和红炉鲜酥的绿豆饼。

    也终于测了下核酸试剂盒；结果是阴性，稍微放心了些。

    虽然心一直有些散，但感觉自己还是对“正念”更开窍了一点。接下来要给自己找一些“心锚”，让自己能快速地回到正念中去。

    January 5th, Thursday #

    白天去附近医院测了下核酸，又送了鞋去洗，然后重新理了行李，打包好了两个登机箱寄回家。

    晚上ETS实验室组会，结束时，曲老师让我、王亮老师、刘佳卉、林泓熠（好像是泓熠）留一下，说要聊采购之类的事。前面组会时，我因为无聊在看小说；留下来后脑子还在小说那边，只记得曲老师问了我何时出国，然后我又脑子一热，跟他建议组里基于OpenAI API开发些“基础设施”用于研究。

    会后我慢慢“清醒”过来，才想起之前佳卉发了几次朋友圈，说被一个同级的、有工作关系的男同学骚扰、被其女友辱骂，正在请系里调解。联想到开会时曲老师几次暗示“不要太在乎讨厌的人”，才意识到说不定这个男同学就是泓熠。之所以额外留下我和王亮，聊了几句又让我俩先退出，应该是曲老师想稍微放个烟雾弹，然后再出面调解，以免别人联想到这件事。而我竟然还在傻傻地和老师聊AI，想想都觉得羞愧。只能安慰自己，在曲老师面前表现得很傻，也许也不全是坏事。

    （但想了想，又觉得自己可能猜错了… 因为佳卉朋友圈截图里的男生的头像，是哈利波特里的罗恩，之前没见泓熠用过 … ）

    而对曲老师提AI，其实也是给自己瞎揽活。我只能再安慰自己，也许是个学习交流的机会（比如拉上刘洋）。

    January 6th, Friday #

    昨晚雨婕探头探脑地跑到我房间，问我有没有空聊聊；我心里了然，借口说没空。今天快速收拾好了房间，把不能带走的行李都寄回了家，然后搬到了亚朵去住，免得需要再和雨婕纠缠。晚上雨婕问，怎么提前搬走了？我说之前混管核酸结果异常，先出来自我隔离一下。她又问，走之前要不要吃个饭？我又说没空。她问我是不是在逃避？我说我这几天已经够忙了，有话直说就行。

    她没办法，只好在微信上说；绕来绕去，问我有没有桃花云云，最后自然还是说复合的事。我看她字里行间满是算计，又看她说“给彼此一次机会”，心里气不打一出来；觉得明明是你只靠嘴说，从来没有过像样的行动，到头来还是给我机会？但既然已经分手，我也没必要拿她当自己人那样掏心掏肺了；只说两人只是不合适，没法走到结婚这一步，希望她以后顺利。

    回想自己这几年浪费的心力和时间，觉得有点可叹可笑；但总归是锻炼了自己，也算值得。

    January 7th, Saturday #

    昨天下午去世纪坛医院和西站的“卡尤迪”分别做了一次核酸；今天两份结果出来了，都是阴性，可以放心。

    昨晚找[[尤培钰]]吃了个饭；今天一起去找[[王心怡]]和[[崔梦秋]]玩密室时，她对昨天吃饭的事只字未提，其中几次明显是充愣。我心想，大概培钰也并没有我看上去那么愣（笑）。之后四个人去梦秋加看电视到晚上，度过了很快乐的一天。

    发现[[王心怡]]的兴趣和性格都和我高度一致；虽然她脑子好到让我无从下手，但这也是戳我的点。

    只是这些都要暂时放在一边了。如果下次回国还有机会，就别再犹犹豫豫啦。

    January 8th, Sunday #

    折腾了二十个小时，终于顺利抵达了亚特兰大。从元旦到现在，飞北京、理行李、做核酸，到今天过了海关，总算是胜利回美。陈鹏把房间按我之前住过时原样布置好了，令我感动不已；于是翻出了很多礼品给他和张薇，聊表感谢。

    January 9th, Monday #

    和馨月聊天，她说我房间的风水不对：不仅容易跟权威起冲突（床坐北朝南），还把滋养都给了前女友（角落放满了她的娃娃）。我心想，简直不能忍；于是花了一整天重新整理、布置房间。折腾到晚上，总算感觉屋子顺气了许多。

    January 10th, Tuesday #

    之前一直担心，如果驾照一直过期怎么办；今天顺利更新了驾照，顿时感觉放心了许多。在美国如果没法开车，实在太不方便。问了Flexcar，说临时驾照没办法用来租车，得等我正式驾照下来再说。我只好托陈鹏帮我租了车。

    January 11th, Wednesday #

    因为倒时差，整个人还是感觉很疲惫；花了一天收拾东西、打扫卫生，总算让房间整洁了起来。

    在桌子的一侧开辟了一个“植物角”，准备了植物灯泡、智能开关，又心血来潮做了个苔藓瓶。还有好多东西想搞。期待。

    January 12th, Thursday #

    今天去学校见了一下Anton，商量了一下这个学期怎么安排。第二篇论文只差开头和结尾就改完了，但他现在要备课，可能月底才有时间。我打算月底之前这两周集中精力把第三篇文章改一遍，下个月再跟他一起改第二篇。同时，下个月打算看一看第四篇文章的证明部分要怎么搞。之前想找个人帮忙搞第四篇，但Anton一直不同意，让我自己研究清楚再说。胳膊拧不过大腿，先弄吧。

    顺便约了一下洗牙，还有测视力。

    晚上回家的时候正好看到李延和王海蓉，就蹭他们的车回家，顺便聊聊天。跟他们聊到和女朋友分手的事，他俩安慰了我几句；又说他们实验室刚来的妹子很好看，可以给我介绍。我自然是不拒绝的（笑）。之后找时间约他们吃饭攀岩好了。

    January 13th, Friday #

    总算是把车租回来了。是个还蛮大的SUV，本田CR-V。开起来还不错。回来的路上逛了一下宜家。把缺的几样家具也买回来了。

    January 14th, Saturday #

    上午跟家里视频了一下。然后去买了菜。下午叫了莱克西他们来家里玩剧本杀，玩到很晚。看大家的反馈，体验应该还不错。之后可以试着多组织点几次。

    January 15th, Sunday #

    这个月第一周主要是在收拾东西、准备飞美国。这个星期基本上就是在安顿。从下周开始。希望能恢复正常的工作节奏。其实从今天开始就想正经干活了，但感觉恢复工作状态还是需要点时间。打算先看看各种公开课视频吧。

    January 16th, Monday #

    曲老师那边要的课程PPT总算实应付出来了。虽然那边应该还有一些工作，但都是下个月才需要考虑的事了。这个月要把学校这边的工作进度努力恢复、多往前赶一点。

    January 17th, Tuesday #

    去和卢老师见了个面，发现这学期的助教工作果然名不虚传，非常轻松。感恩（笑）。

    January 18th, Wednesday #

    去学校的牙科诊所检查了一下。毕竟三年没洗牙了；这次检查完，医生说需要再深度洗牙。我既怕钱包疼，又怕牙疼；但是没办法，只能按医生说的来。

    晚上和雷雷吃了个饭，相互update了一下情况。

    January 19th, Thursday #

    回来之后还一直没怎么改第三篇文章。今天要去见Anton，临时抱佛脚，只能又去跟他聊收敛性证明那个东西。聊完之后觉得，其实这样没有意义，还是要一次把精力集中在一件事情上。目前的主要工作是先把第三篇文章整体改完，然后再谈其他的问题。

    January 20th, Friday #

    回美国来快两周了。之前折腾的水草缸和苔藓瓶。最近总算稍微有一点样子了。其实会担心。不知道我自己在这个地方、乃至于在美国能待多久？但想来想去，还是要保持一个良好的生活质量。不能因为未来有一些不确定性，现在就连日子都不过了。要积极向上、有生命力一点。

    January 21st, Saturday #

    过年了。早上起来跟家里视频了一下，也给一些朋友发了微信拜年，然后就跟陈鹏和张薇去大中华买菜。中午吃饭的时候，张薇问我是不是还有可能跟雨婕复合。我说应该不会了；她就说，可以给我介绍他们学校的女生认识。听她对这个女生的描述，总觉得跟雨婕有点像，让我有些“PTSD”。不过有这个机会，当然还是先见见的好。

    想起来之前李延和王海蓉也说，要介绍他们实验室的女生给我认识，不过到现在还没有下文。之前约雷雷一起玩儿剧本杀，现在他也还没有再问。社交不容易；闲棋冷子，慢慢布局。

    晚上陈鹏叫了[[徐晨妍]]、[[郭涵一]]、[[丁烈骁]]来家里吃饭。饭桌上大家一直在聊三体；吃完饭后女生们去看春晚的唱跳节目，我就和丁老板聊了一下人工智能的事儿，他果然也发现，现在人工智能是个最大的机会。他讲了一些观点，让我很受启发。我倒是没和他说太多我的想法 —— 总感觉说的太多，行动的动力就会变少。也借机跟台湾室友Frank聊了几句，算是增进了一点了解。

    January 22th, Sunday #

    今天去办了攀岩卡，见到了[[贾晓彤]]、[[张慧]]，以及三个和她们一起在攀岩的男生。本想跟张慧多套套近乎，但是没找到什么好机会，只能之后再想办法。不管怎么说，还是先回到攀岩这个圈子来，从这个渠道也多交一些朋友吧。

    回到美国两周了，这一周的工作状态比上一周好一些，算是开始正常工作了。不过效率还是很一般。

    January 23th, Monday #

    这两天有个来特兰大开会的德国教授给Anton发了一封邮件，说想见见他和她的学生。于是我、[[刘心语]]、[[施晓晨]]今天上午就被Anton拉到学校，一起见这个教授。后来才知道，这个教授是想随机social一下，但他做的方向跟我们几个没多大关系，整个聊天也比较尴尬。我倒是借机会跟晓晨和心语都聊了聊天。也不算白来一趟。

    其实更早的时候还想参加一个文化班的聊天活动，所以我很早就起来，先到了学校。但是感觉这次聊天的内容有点插不上话，说了几次话也都有点冷场。所以后来就索性不说话了。下午回家感觉挺困的。就睡了一觉，睡醒了，已经快晚上了。陈鹏说想去Costco买菜，我就搭了顺风车去了一趟PetSmart买了点水草往鱼缸里种。

    睡前研究了一下苹果手表的睡眠记录功能，准备用起来调整一下作息时间。

    January 24th, Tuesday #

    早先看见某公众号在推一个叫NICO的SM圈交友APP，就注册了一个账号。上周试着勾搭了一个23岁学计算机的小姑娘，但是两个人聊天比较困难，也没有什么共同话题，只好不了了之。昨天又和一个28岁的“知性恋”姑娘聊了天。她是在学SCAD艺术的，在做服装设计师，同时也兼职做服装系的教授。两个人都比较热情，也都互相有好感，聊的不错。

    好奇她的长相，就去Google上“人肉”了她，查到了他的Instagram，也看到了他的中文名[[尔若焰]]。后来在微信上跟她说她的名字好听，她被吓了一跳；再聊天，她说感觉有点被冒犯，也没做好跟陌生网友互相披露自己的准备。她说，因为这样的情况，她可能不再会跟我发展任何两性关系，只做普通朋友好了。我有错在先，不好多说什么，至于能不能挽回；等之后再看吧。

    因为这件事，外加一整天喝了很多咖啡和茶。整个人精神很亢奋，很晚才睡。

    January 25th, Wednesday #

    昨天大概凌晨两三点钟才睡着，睡到今天早上九点多；醒了之后状态还是很差。上午一直在刷视频，刷到中午吃完饭，又回去睡了一觉，睡到下午三点才起。起来以后收拾东西，把整个屋子仔细打扫了一遍。然后把床单被罩也都洗掉了。又跳绳锻炼了一下，然后做了晚饭，和陈鹏一起吃了。饭后休息了一阵子，感觉状态好转了一些，不过还是比较疲劳。

    更早的时候不好说，不过从周一到今天这三天，我的效率比在北京、通化的时候差得多。直接的原因当然是不够自律，比如作息混乱，缺乏锻炼。比较纵欲等等；但更深层的原因应该是心理上的。樊登读书里面有本书说，自律需要很高的自尊水平才能得以实现。我个人的感觉是，在北京和通化的时候，尤其是有家人朋友陪伴的时候。整个人的状态和效率就会好一些。来到美国后，最近两周一直感觉比较孤独；内心有一种躁动，需要设法填补，所以不太容易集中精神。

    但是建立朋友圈子呀、找对象呀、有性生活呀，都不是马上能搞定的事儿，只能慢慢来了。

    January 26th, Thursday #

    这两天又和[[尔若焰]]聊了不少。昨天她提议出来见一面、吃个饭；我答应了，开车接她去一起吃了个拉面。应该说，她的硬件条件真的很好：是学校的兼职老师，事业发展很好，在艺术领域已经小有名气。但她的性格、身材乃至于生活习惯，实在是不太对我胃口。虽然两个人聊到很晚，但我内心其实不太打得起兴趣。

    January 27th, Friday #

    美国的”原计划“是快速搞定俄罗斯，重新控制欧洲盟友，获得政治经济收益，然后转向”下一场“台海，如法炮制。但现在看来，美国在俄乌战场无法达到目的，甚至已经难以脱身，因为各集团利益牵扯太深了。另外，很多美国及各国精英认为，如果美国无法搞定俄国，那也就更无法搞定中国。如果俄乌战场失败，那台海问题上，日本、澳大利亚敢于下场的可能性就变得很低。这也进一步构成美国现在难以脱身、承受失败后果的原因。事实上，从一开始，美国对这次战争的政治目的就不够明确；只是宣称要”彻底击败、颠覆俄国，甚至审判普京“。这样的目标是不可能做到的。

    北约东扩对美国而言，其后果是灾难性的，可以看做是短视的资本利益集团绑架了美国的长远利益。北约东扩的其中一个后果，就是将俄罗斯彻底推向中国。于是这一次，中国意外地成了俄乌战争赢家。考虑到俄乌冲突对美国及其东亚盟友的影响，台海冲突的可能降低。；另一方面，俄乌冲突也对中国是巨大震撼，也会使得中国处理台海时更谨慎；除非美国明确支持独立，使得中国别无选择、被迫全面行动。理智的美国总统不会这样，不会给出明确承诺，而是一定会保持战略模糊；但前提是美国政治真的还能足够理智。

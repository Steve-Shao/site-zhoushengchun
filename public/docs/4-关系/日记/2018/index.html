Journal 2018 #

January #

Plan #

新的开始，最要紧的事还是自身的修行。生活习惯要坚持改善，待人接物也要更用心。

八号开学，这个月上旬有不少杂事要处理。月中要翘一周的课去夏威夷玩，是这个月最期待的事情。旅游回来，应该还能赶上狮子王歌剧的巡演。

这个月的重心在课业和研究。新学期的两门课都不轻松，加上翘课，应该要下不少额外功夫。研究方面，之前的论文稍微放一下，要先把Anton给的新课题研究清楚。

量化投资方面，之前一个月里尝试了很多模型，但效果不佳。而我也更多地意识到，自己在基础知识方面还欠缺很多。这方面的研究会先放下，耐住性子慢慢用功。

唯一觉得能用的策略，是“养老金”配平模型；于是打算结合长线投资来使用。上线这个模型，也是这个月月初的任务之一。

Summary #

第一周总结了之前量化投资方面的研究。到暑假之前，暂定只测试“养老金”算法，而不再铺开新的工作。

第二周开学，忙了不少事情，大致上安排了这学期的学习和研究工作。

第三周去夏威夷旅行。

第四周回到学校，补作业和课程，赶之前落下的进度。

第五周状态大提升，开始做办公室，高效率地学习和研究。

一点期待：之后每个星期，都该比前一个星期有明显的提升，而尽量不要有退步。提升可以是在习惯方面，也可以是在与人交往上，或是其他。

助教的工作做得一直不好，月底更是忘了和Amanda的见面。之后要勤勤恳恳地工作，不要给再别人添麻烦。

话说这个月一次日记都没落下！表扬！

January 1st, Monday #

今天晚起，收拾完东西、做好饭时已是中午。不到一点钟出发去帮同舟搬家，到六点半才算弄好。晚饭去吃了一直想吃的川渝火锅，味道很好，只是猪脑实在不敢碰 (笑)。饭店很远，往返时间久，最后到家已经十一点了。

天气极冷，晚上开车回家，忽然想起刚进Gimme时，雪天凌晨出外勤，冻得撑不住；想想已是两年前了。两年来摔摔打打，果然身心都健壮了许多。

希望又是更好的一年。

January 2nd, Tuesday #

下午把猫送回家，和易琳他们吃了个晚饭。晚上回家，客厅空空，稍微有点落寞。以后如果买了房子，也许也会养只猫。

今天写了一点Hitbtc的API，稍微测试了下，觉得rebalance算法的收益并没想象中高。不过总归要慢慢来，先把技术问题解决，再看看能不能提高收益。

这两天慧晗过来，加上各种杂事，以至没太来得及细细规划。明早要研究研究开学的计划了。

January 3rd, Wednesday #

昨晚失眠到近凌晨四点，难受至极，今天一上午昏昏沉沉。中午出门和陈依伦等吃饭，之后回来补觉。下午去健身房练背，希望能帮助身体调整睡眠。

失眠的原因是一直在想投资的事，于是决定尽快上线算法，免得自己继续瞎想。考虑到Hitbtc交易量较低、币种不足，晚上便注册了Binance。谁知注册之后，竟无法生成转入地址。研究了半天也未能解决问题，随后看到了系统公告，才发现这也许是系统维护的原因。

冷静下来后，忽然发觉市场上BTC与各主流币震荡、山寨币飙高 —- 有些像是资金在逃窜的样子。又看到中国限制比特币矿场的消息，这才有些害怕。也许是天意教我今天没能转入资金，也许就避开了一次大跌。

不论怎样，开始实盘投资，也是为了更多地把精力放回到学业和研究上。希望这两天能找到合适的机会入场；而之后，就不时地配平、看新闻，。短期内不再花太多时间了。

January 4th, Thursday #

从Hitbtc提币到Binance，却一直pending了几个小时，眼睁睁地看着ETH慢慢跌下去，甚至不知道还能不能把币找回来，气得想哭。不知是不是上天在用另一种方式告诉我“还不是时候”，这种方式也未免太揪心了点。

要是Binance昨天能正常用，昨天就转账了，也不会遇上这事。心里真堵。

想想这两天被这事闹得都没怎么干正事，得收收心了。明天有TA的培训，要记得。

January 5th, Friday #

于是终于开始在Binance上投资了，共挑了十个币种。虽然知道再等等的话价格很可能大跌，但终归没忍住。因此，恐怕要做好近几周内投资被腰斩的准备。希望能保持平常心：不要因为心急而频繁操作、胡乱操作，也不要整天盯盘浪费时间。

本想写个API和网页应用来算数字，但在Authentication的部分一直没弄懂，只好作罢。同时也发现了很多问题，尤其是价差、币价变动太快，造成仓位很难配平。因此，最好还是只针对较大变动进行操作，买卖都要谨慎。

明天要和家里视频、买菜、去张焱家吃饭；后天要去教会。同时，这两天也要解决各种杂事，做好开学前的各种准备工作。从周一开始，就要把弦绷紧，好好学习了。

January 6th, Saturday #

其实今天本可以更充实的，因为并没去买菜，也没做拉伸；结果过得有些无所事事。

昨天开始投资，从2100起，今天已经接近2300，让我更怕了… 总之还是要平常心：大市别想多，专心赚羊毛钱。

要尽快收心。

January 7th, Sunday #

本来明天第一天上课，但今晚忽然收到了停课通知。学校解释，这是因为冻雨和明晚的橄榄球赛。又看到了特朗普名为会去看球的消息，忽然想起昨晚开到张焱家旁边的军用机场时，出现了警车和八辆大巴车混杂的车队。想必特朗普昨晚飞到了那个机场，而车队是去接他的了。

今晚慧晗忽然要我替她买三个莱特币，结果刚买完就一路下跌，有点揪心。要是接下来真的暴跌，之后无论是否向她要钱都很纠结。唉，觉得费力不讨好。

老实说，要是明天不停课的话，我的时间已经相当紧张了。既然被给予了这个机会，就老实抓住，好好干活吧。又，明天冻雨，也许没办法去CRC，打算早上在小区健身房健身了。

January 8th, Monday #

开学第一天，并没上课。白天去买了些东西，回家后处理了一堆TA相关的杂事。晚上本想预习一下课程，终于还是水掉了。

预感这学期TA的工作会很忙，毕竟每周光网上答疑就要八个小时 … 想到下周还要出去，顿时觉得很担忧 TwT …

January 9th, Tuesday #

昨晚盘算了一下，春学期最重要的任务还是发论文，以及开启新的研究，而课程是其次。暑假可以多花时间研究金融、投资和编程的知识，把自动交易机器人搭起来。当然，还要留出一个月准备资格考。

想来想去，时间蛮紧的，于是退掉了PHIL 6000和ISYE 8803，以便集中注意力。

昨天弄了一些TA的杂事，花了不到两个小时；而今天上了课、弄了车牌和免税表，花了大半天时间。开学之初，弦也渐渐绷起来了。不过没锻炼，算是休息一天。

关于毕市的一个预测：长线上会阴跌很久，短期上会有闪崩。当下还是决定不撤资，并且这几天想准备一点钱，在GDAX上市场价下方1/3～1/5处抄底。

January 10th, Wednesday #

今天到没什么特别的事，只是过得还是比较散漫。

终于去上了实分析课，但内容和我想象的不同，更接近测度论的内容；而这大约也是Anton希望我学习的内容，这样来看，之后该花点时间补上基础的实分析。好在手上有陶哲轩的教材。

明后天晚上有TA的工作，出去玩之前已经不太有实现系统地学东西。希望明天能把下周要学的内容尽量熟悉些。

January 11th, Thursday #

晚上做了这学期的第一次TA工作，大致上就是在网上论坛回答问题，以及帮在线讲解的人回答问题。折腾了一整晚，还是蛮累的。

老实说，我真的不大喜欢TA工作，而且做这个也真的比较费时间。因此，暑假打算认真地试试申请科研经费。不过就这学期而言，还是会认真地做的 – 毕竟，再也不想看到上学期末的那种学生评价了。

从5号重新投资数字货币以来，资产先暴涨了一下，之后就回到2100的成本附近挣扎。估计或早或晚，资产会跌破这个成本的；心理上要做好腰斩的准备。

其实不知道这种“重新配平”的策略是不是真的好过装死。有空的话，可以写点代码看一看。

又，在夏威夷期间，打算抽时间看看Anton之前给的研究课题；为此明天要做些准备。

January 12th, Friday #

明天就要出发去夏威夷啦，心里还是挺激动的。于是主要的任务还是完好；在此之外有可能的话就减减肥 T_T … 再有可能的话就看看论文 TwT …

现在体重又回到85kg了；自从去年二月然飙升到这个体重，就一直没能有效地控制下来。减肥这个事情必须重视了(恶狠狠地)，尽快回到75标准体重。因为刚刚收拾行李的时候发现，好多喜欢的裤子都穿不上了…

虽然出去玩的话比较担心课业，但总归得一样样来。好在先把6663的作业写完了，实分析的作业等到下周回来再写好了。旅游期间抽空看看新的研究课题。

刚刚算了下银行账单，发现还是比较收支不平衡的 … 记得开学前checking账户上有4000（心理上以为是5000）结果到现在还是4000。然而刚刚买衣服和吸尘器花了500，还上所有剩下的信用卡账单需要500，而夏威夷之行保守估计还要1000块 … 于是实际余额只有2000了…

虽然各种利息算上能挤出400，但觉得杯水车薪 T__T .. 况且今年暑假大概不会实习了，于是又是没有收入的三个月。希望投资能有些回报来缓解钱包的紧张吧，虽然心里知道其实不太指望的上 …

忽然觉得有点惨嗯 …

January 13th, Saturday #

抵达夏威夷，落脚/买东西/看日落/吃完饭，现在虽然只是晚上八点半，但由于五个小时的时差，实际上已经困得不行。

听说今早有夏威夷的政府工作人员误操作，结果发出了导弹来袭警报，全市一片混乱。想想万一自己真的遭遇导弹来袭，搞不好只剩下给家人打电话的时间，心情有点复杂。

嘛，还是希望能好好休息一下，明天才能好好玩。

January 14th, Sunday #

夏威夷比我想象中美得多；怎么说呢，有种直抵内心的冲击力。早上去了一个山顶的Lookout，半个岛屿的景色尽收眼底，心里就忽然爱上了这个地方。

下午去沙滩，发现自己果然不懂怎么在海里游泳；不仅不懂踩水，对风浪也是很怕的。诶，海水真是咸。不过对于自己敢下海游泳这件事，还是有那么点欣慰的。

后来去礁石区找小鱼小虾，结果被小孩子包围了；感觉有点羞愧，于是快速逃开。

另外，总觉得又开始不过脑子乱说话了，请把嘴管得严一点 (笑)。

January 15th, Monday #

之前发邮件帮张焱refer在WestRock的工作，今早转发了他的简历。如果能成功，自然是好事。只是真的跟Manish说了之后，心里有有点虚。有点担心这样做是短了自己的机会（因为没说自己去不去却先推荐了别人）；若是张焱不能成功，我就是白白损失了。

不过现在想这些也没什么用就是了。

今天玩得也蛮开心：参观了珍珠港，爬了两个Trail，走了像是加州一号公路的环岛路。环岛路上停车逛了一个礁石区，还看到了海胆。

十点钟才回来，疲倦的不行，还要收拾东西，以便明天早起去机场。

晚上回来发现国内传出政策，以至比特币开始大瀑布；不知这次回到何处了。倒也是机会。

January 16th, Tuesday #

一大早起床赶路，飞机汽车，终于到了在大岛上的落脚处。其实没做什么耗体力的事，但意外的疲惫；大概是前两天玩得太用力吧。

下午三点出发去浮潜，发现自己果然很怂，到最后都只敢在刚过膝盖的小沟里转悠。不过还是很欣慰自己终于敢下水了；也算是为明天做了个准备。

吃完饭到家又是将近点钟，这几天也许真的不会有时间看论文了，看来回去要拼命一波 …

两天来投资已经跌到1600了，其实是有点慌的 0__0 …

除此以外，这还是玩得很开心的；每次都累得想立刻睡觉，但每次都很期待下一天。

January 17th, Wednesday #

早上起来投资已跌破1400了；想着到1000就加仓，但10号的转账至今未到，有点悲桑 TwT .. 之前给慧晗买莱特币倒是立刻到账了，不知之后要不要也直接去买。

上午去浮潜，仍然在潜水海湾游荡；但这次运气很好，鱼很多，还看到了海龟；而其他去深水的人反而没有看到 (笑)。之后去爬海拔四千米的Mauna Kea看日落加看星星。虽然有点高原反应，但景色简直像是在外星，体验极其值得。

January 18th, Thursday #

一上午退货换车还浮潜装备，然后装车出发，沿着大岛南侧千万东边的另一个住处。途中路过South Point，景色极美，草原、悬崖和海浪融为一体，让人觉得什么都值得了。只是风沙太大，到最后满脸满嘴满身都是土，眼镜也雾蒙蒙一片。

之后路过黑沙滩，捡了几个椰子，暂时没想到怎么把它们弄开。晚上吃了广东菜，然后到住处落脚，玩猜谜游戏到深夜。住处一如预期的好，很有美式与泰式装修风格结合的感觉。熟睡一觉，准备好明天去看岩浆。

夏威夷真的是天然的游乐场：有繁华的城市，有历史遗迹，也有很美的爬山路线；有珊瑚礁、阳光沙滩，也可以在像是一号公路的地方看海和礁石。除此之外，这里还有温带的森林和草原，有热带的荒漠和雨林，有火山和雪山，等等等等。这次花了一周多，也只走了两个岛的一小部分；有太多值得期待。

忽然觉得一直蜷缩在亚特兰大的一个小角落里，竟是那么悲哀的事情。

希望能更多地走出来，体悟这个世界。当然，也希望尽快地在财富上和时间上变得自由。

January 19th, Friday #

今天的主要内容是看火山。

上午去火山公园，看到了火山口和雨林；下午直接爬岩浆岩去看岩浆。路极难走，往返整整十公里，全都是锋利的乱石。除此之外，还要外加往返六公里的骑行。去程到一半，忽然刮风下雨，到了目的地才停。回程时天色全黑，不仅手电光昏暗，体力也渐渐下降，非常辛苦。

但是值得。在三米的距离上看流动的岩浆，这种体验太难得。

心里暗下决心，等回去以后，也要这么一直用力地生活。

又，由于大雪听课，以及Erera的安排变化，这周竟然就只翘了一节6663，开心！

January 20th, Saturday #

上午闲逛，看了些观景台；下午飞檀香山，吃饭入住。明天要起很早，坐飞机回亚特兰大。

今天最有趣的部分是徒手开椰子：椰子是熟透的那种，外面完全木质化，表皮坚硬如铁，里面是既有韧性的纤维。椰汁撒了大半，

吃到时才发现，越熟的椰子椰汁越少、椰肉越多，而椰肉也变得越坚硬。外壳发亮时，椰肉还在可吃的状态（硬而有嚼劲，很香）；之后就变得更像木头了。

旅游像是生活的刷新键。到旅途的最后，感悟很多，也充满了想要改变的想法。回去后慢慢整理吧，也要想想陪隐形眼镜、整理日记、整理投资信息的计划。

January 21st, Sunday #

昨晚少睡了些，今天起大早赶飞机，先飞洛杉矶。转机时得知后面的航班延误，多等了几个小时。现在来看，恐怕要凌晨三点或四点才能到家。

昨天睡前，久违地收到毕芸的微信，问过年是否回家；于是就聊了几句。看到她发来自己的照片时，心里忽然悸动得不行。

她真美啊，大概是和我谈过恋爱的女孩中最美的一个；虽然常驼背，其实身体也完美的不行。现在回想，她的性格有些别扭，但也恰让我有了更多冲动。

这大概就是初恋的情结吧，那时真的一片真心，到现在也被影响着；可那时也真的太傻，总是做各种蠢事。

而淳映正相反，虽然对我并无身体的吸引力，却有着令我欲罢不能的危险性格。那种聪明、果敢、独立而又难以捉摸的脾性，让我深陷其中，真正体会到了爱情与失去爱情的滋味。

国内路上的交通，街道上的车子和路牌，大处小处的城市规划，已经被不少学者指出了问题。亲戚家的室内装修和设计，在我看来也是问题多多。生活习惯上，好多事情注意了就没事，却少有人注意到。而我、我父母，乃至更多人，明明知道却难以坚持。

难以坚持，也有部分是因为，我还不明白“坚持”所需要的知识。

改变自己，言传身教，任重道远。

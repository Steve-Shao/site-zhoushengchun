    宗教体验

    日前和朋友聊天，说起自己饿了一整天后，深夜里吃到一碗豆腐汤时的感受。“简直是种宗教体验”，我这样形容。

    一句话让朋友笑了好久。“真没出息”，她说。

    我不以为然。直到几天前，我才读到这样一段话：“宗教体验，是人遭遇到远超自己认知的事物，并产生为其付出一切的冲动时的感受。”

    看来，说宗教体验是太过了；一碗热汤能带来的，最多算是种艺术体验罢了。

    其实就算艺术体验，也是相当了不起的东西；好像听到一段美妙的旋律，总是不由得想要拍起手来。大到雕梁画柱的宏伟建筑，小到深冬时咕噜咕噜的小火锅，都能给人一种艺术体验。“原来世上还有这么美好的东西”，大体上就是这样的感受；可其中总归没什么付出的冲动。非要说的话，占有的冲动多一点。

    能带来宗教体验的东西，要厉害得多才行：是一望无际的大陆和海洋，是广袤而苍凉的无人深空；又或者是纵贯千年的辉煌文明，是其中灿若群星的伟大灵魂。宗教体验中，应当是夹杂着一种自惭形秽的：如果自己是那样的渺小和卑微，付出一切何必犹豫。

    然而这只是对于我；若是寿司之神那样的匠人，想必是能从一粒米中看见天地众妙的。每个人因经历、境遇不同，感知力也各不相同。艺术与宗教的体验终归可遇不可求，但读书行路、克己修身也许是条捷径。

    宗教体验常常是召唤的钟声；伴随着虔诚的付出，人生也就被赋予了意义。对我们这最迷茫的一代来说，这钟声未尝不是一股清流。而我们的迷茫只是这个文明的迷茫中的一小部分：如今的人类文明无父无母，能被歌颂的，只剩下人类本身。

    人生本无意义。既然没有人愿意点破，听从内心的钟声也不错。

    Jan 25th, Wednesday #

    远方炊烟

    明明已有两年没回家乡，一切却熟悉得不像话。楼下的菜摊好像昨天才去过，可老板早已认不出我。依旧能和高中同学打牌聊天，但话题渐渐变得不太一样。从杭州到美国的七年，忽然变得像是一场梦。

    只有身体早已适应了那份温暖湿润；干燥而凛冽的空气混着烟尘，让我从喉咙肿了到牙床，疼得几乎张不开嘴。

    东北的冬天很急躁。大概是冷风太盛，把大家都吹得一肚子无名火。带着这股火，不仅车子横冲直撞，连行人也毫无畏惧。开在这样的路上，我们一家也不自觉地相互抱怨起来。

    回乡下探亲是大工程。全家一大早就开始装车，一直装到中午，连后座都被塞满，只留下一点点空间坐人。泡沫盒子顶得车门几乎关不上，最后只好用踹的。

    新修通的高速路平整极了，却在最后一小段戛然而止。取而代之的是窄窄的双向车道，上面的车子、摩托和行人都挤在一起。天色不到五点就漆黑一片。两侧的车子都拼命地闪烁着远光灯，驱赶着隐没在黑暗中的人群，宣泄着各自的愤怒。

    漫长的旅途后是饭桌上吵嚷的亲人。年龄加上酒精，让叔伯们变得宽声大嗓而又絮絮叨叨。杯酒下肚，更有从未见过的亲戚过来聊遍大道理，替你安排好未来几十年的人生。幸而婶子依旧那般干净利落；欣慰之余，又哀叹起时光，哀叹起自己的格格不入。

    走到院子去吹风来让自己清醒，忽然看到满天的繁星，清澈而透亮。在那下面，无边无际的大地上绵延着万家灯火。炊烟钻进鼻子里，脑海最深处的记忆瞬间苏醒，心情便一下子复杂得不行。

    明天一大早要去祖坟；那时叔伯会烧起纸钱、放起鞭炮，告诉祖先们自己的子孙是多么地有出息。之后要回去城里的外婆家，去体验另一番新年的光景。再之后，整片大地上会有无数的烟花炸响；那里有着千家万户的百味杂陈。

    回到屋里，家里人在聊着最平常的话：粮食一年贱过一年，几乎已抵不过本钱；养猪养牛太辛苦，还要担心染病赔本；村里人赌钱被骗，输得倾家荡产；亲戚的拖拉机龙骨突然断焊，重伤住院了一个月 …

    而在这日常的苦难之中，也有付出得到回报的喜悦，有对未来的期盼，有新生命的降生。那些已经不再懂我的人，也依然会叫我去珍惜天上的雨、地上的粮食，去珍惜老天爷的一份又一份赏赐。在温暖的火炕上，无论隔阂，大家都是在拼了命地活着；为自己，也为别人。

    Feb 10th, Friday #

    三个星期很快过去；刚适应家里的气氛，又要投入美国的生活节奏里。除了在新年和家人团聚，旅行带来的清新感也令我十分愉悦；这种感觉说来，就像是生活被点了刷新键一样。“看来以后也要四处多走走”，我对自己这样说。脑子里纽约、加州和上海的行程，也可以稍微计划起来啦。

    虽然现在这样清新，在家的日子可是乱七八糟。除了一点点工作以外，基本上就是吃、睡和玩。到最后，总觉得一直在搪塞公司和教授那边。而对应地，那两边也现在几乎联系不上，弄得我心很慌。对家里人，也没能好好做点什么；尤其是对二姨夫那边，还是没有认真地教教弟弟学习。

    不过还是好事更多：其中最重要的，莫过于听说博士有戏了。那晚好久没睡，想了很多长远的事，想了很多自己的责任，一夜间又成熟了些许。这种成熟，刚来美国时在颜妍身上看到过。现在到了自己身上，自然很让我愉快。和婷婷及其他朋友的聊天令人开心，和爸妈的聊天也让人很舒服。看着老爸说着自己未来几年的计划，有几分像是个老地主，总觉得很有趣。

    所以现在的状态，说是“不确定”更准确些。

    现在正坐在北京飞向西雅图的航班上；踩着前一个OPT的截止日期入关，后面的EAD卡又还没有寄到，能否顺利过关自然是不确定的。离开了三周，和公司、教授都缺乏联系；现在两边的情况，也有许多不确定。公司的工作和论文还不知道什么时候能够收尾，之后到开学前的时间还在安排中，也算是不确定。自觉心中喜欢婷婷多些，但对方的想法，以及和慧晗的关系，也都在不确定之中。

    要面对这种种的不确定，心中要有份平安才行。有平安才不会被焦虑烧毁，也不会陷入惰性的泥潭。这平安应当不仅是外松内紧的生活，更是相信着自己会被给予正确的道路，以及能把这道路认真走下去的信心。

    假期的安排，也要好好计划下啦。

    Dec 16th, Sat #

    昨天折腾慧晗的毕业典礼和聚会，外加练背；很难说是愉快，但所幸还过得去。到昨天晚上，才觉得疲惫的不行；于是大睡十个小时，痛快的很。

    Dec 19th, Tue #

    人在冬天容易疲倦；昨晚早早上床，竟又一觉睡了十个钟头。前两天研究股票算法，加上忙活各种事，过得蛮充实；只是今天上午有点晕晕乎乎，到中午才开始干活。

    Dec 24th, Sun #

    今天是平安夜，却一天都宅家看论文，觉得有些凄凉。其实并不记得去年此时在做什么 – 但记得前年此时，是和梦莹有了人生中的第一次；不觉有些怀念。

    说到底，谁也怨不得，唯有自己振作，时时不放松。明天起会认真生活，愿有个新的开始。

    今天是平安夜，祈祷自己会变成更好的人。

    上午和下午开发算法，思路稍微通了些，明天可以实施出来试试看。中午出去买菜吃饭，吃得很是舒服。晚上给张焱讲WestRock的事，之后会尝试发邮件推荐他；之后和张焱老陈打游戏，打得很是痛快。

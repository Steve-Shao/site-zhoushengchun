January #

January 1st, Tuesday #

久违地充满仪式感地跨年。虽然昨晚熬到两点多，早上还是早起去学校换了新的日历本。前几天定了许多新年计划。这样认真地对待元旦，还真的有点像小时候。恢复健身也将近一周了，又进入了“身上总是有哪里在酸痛”的状态。我还蛮喜欢这种状态的。

下午送走了王文佳和李易琳的两只猫，稍微有点舍不得。吃饭时听李易琳说，双色布偶猫光是猫仔就要一千五百美元一只。原来这笨猫出身这么高贵，之前还真是不识货 (笑)。又想到，如果养着么一窝猫，让他们放开了生，岂不是一年进账几万美元 (误)？

昨晚和小白、老陈在张焱家跨年，包饺子、做菜以及打游戏。跨年的时候在看励志拳击电影“warrior”。看来，2019也许会是非常猛烈的一年了。

January 2nd, Wednesday #

希望新年能养成做读书笔记的习惯。以前觉得做笔记很累，做出来也很零散。现在发现不必自己写，只做摘抄，注明出处即可。对问题有了足够的了解之后，就能把摘抄的段落分类，整理出清晰的知识框架来。这样，读书笔记自身就成了一本书。

好的段落，不仅可以从书里摘，也可以从电影、电视剧、博客文章、演讲里摘。如果笔记的完成度足够高，甚至可以做成有侧边栏的网页文件，放到网络上去共享。之前做了一些经济学的笔记，参考了很多政府文件和黄奇帆的演讲。希望接下来能把这个笔记好好完成。

我在文字方面的功底还不够，也缺乏组织大篇幅文章的训练。但我对自己的总结归纳能力、记忆力都有自信。用做摘抄来代替自己写总结，可以充分地扬长避短，也可以节约时间。

以前读书，常常在许多本书里跳来跳去，读得乱七八糟。抓住某一个点，多做些对比阅读是好的，但还是要有个主次，一本一本地读完。目前在看英剧“Yes, Minister”。欢笑之余，可以试着做一点对英国内政外交的思考，作为头脑锻炼。发觉自己越来越像个absent-minded professor了。真不知是好是坏。

January 3rd, Thursday #

新年伊始，有很多杂事要办。花了一上午四处“要钱”，但折腾了许久，目前还没要回任何一个。首先是INFORMS的报销。因为管财务的是新人，之前闹出了各种状况，以至于到现在钱还没到，只好月底再看。然后是州税的退税，过了九个月，竟然还毫无踪影。打电话过去也各种没人接，也只好晚些再看。再就是想把Chase投资账户里的钱放回存款里，但管投资的人也联系不上。自己试着下单，但没反应。很气。

下午去换健身房的柜子，也被告知明天再来。学杂费倒是顺利交了。交钱总是更顺利些。

虽然还在放假，系里已经有不少人来了。久违地见到了陈依伦，聊了半个多钟头。打算把下午的时间用来学Tensorflow，毕竟刚刚找到感觉，要趁热打铁。越是不顺利，越要努力向前。

January 4th, Friday #

之前两天一直对Tensorflow如何工作很迷惑，到今天总算有了些感觉。不过优化算法出现“nan”的问题还是没有解决，也没搞清楚为什么会这样。看来之后还要下更多功夫。

昨天下午卖出了Chase投资账户里的基金，今天下午更新了车牌。剩下的事情里，CRC的柜子应该再等等就可以弄，INFORMS的报销月底应该也能弄好。唯独州税的退税，隔了这么久没来，政府部门的电话也完全打不通，毫无办法，只能干生气。也许之后会跑一趟州税办公室试试看。

还是要有这种“贼不走空”的精神：无论顺利或不顺利，总归要每次都得到点成果回来。之后重整思路，再布置新一次的进攻，就可以离胜利更进一步。

中午去和熊吃饭，得知他打算回国过年。在那之前，没有再回亚特兰大的计划。

饭后去了理发，可以以清新的面貌面对新的学期了。

January 6th, Sunday #

这周周中五天都比较充实，但周末两天过得很糟糕。而且由于连续熬夜失眠，精神也很差。

新年还是要有新气象，更何况明天就开学了。这两天的情况，以后最好不要再发生。

January 7th, Monday #

早上收到Julie Smith的邮件，说INFORMS的报销里，由于错过了21天期限，学校不会付那500美元的部分。在我连续地追问下，她才承认自己根本没有跟校方解释情况，然后跑去解释了。上周才刚刚和我道歉，说报销出问题，自己有过失；这周又马上变回一副又懒又蠢的样子，让人无奈。说到底，正常人会三分钟里五次问人家姓什么么？

多亏了那一通灵魂追问，刚刚总算收到邮件，说情况已经解释清楚了，校方会付钱。毛主席说斗争是团结的手段，要“有理、有力、有节”地作斗争，此言不虚。

开学第一天，本以为能集中看看资料，但时间几乎都用来做杂事了。除了报销的事，先是给Anton发了邮件，约了这学期每周三见面；然后去见了助教的老师Gunter Sharp，谈了一下要做的工作。至于柜子的事，CRC这边管事的人依然神龙见首不见尾，有点心累。

January 8th, Tuesday #

大概是昨天重新开始做力量训练的缘故，昨晚一觉睡了近12个小时，睡得天昏地暗。睡个好觉倒是好事，但把今天的高维统计课睡过了，这就有点糟糕 (捂脸)。

早上看到“饭统戴老板”的公众号更新，是篇和“华尔街见闻”合作，采访电子行业分析师赵晓光的视频和文章。对于文中提到的行业分析方法，我十分感兴趣。但优先级不足，所以先记下。

January 9th, Wednesday #

新年来已经一周半。工作方面，前一周在“找状态”，后半周在“找方向”。下午和Anton见面讨论之后，总算确定了方向，可以全面开工了。

由于需要证明的东西太多太杂，决定不再在第一篇论文里讨论Stochastic Optimization的话题。如果之后有新发现，可以单独作为一篇文章发表。因此，也不必再把学Tensorflow作为主要工作，只当作业余娱乐就好。事实上，我现在更倾向于用Pytorch。

上周六，熊子旋的同事张硕来找我，想转发他们的一个招聘信息。我粗心大意，没问过熊就直接帮忙转了；事后才知道熊根本不知道这件事。不分远近亲疏，大概给人留下了不好的印象。和熊聊过，才知道张硕在公司很少干活，都是熊帮忙做。现在没人帮忙，又不敢和熊说，只好自己悄悄地四处找靠山。之后几天，张硕果然又跑过来，有一搭没一搭地打听ISyE系里的情况。、

社会上像这样靠“寄生”过活的人，不知有多少呢？

January 10th, Thursday #

早上起得不算晚，但比较拖沓，以至十点才到学校。健身房的柜子终于换好了。除了没有挨着镜子，其他部分都满意。会议报销又出小插曲，继续发邮件(委婉地)紧逼。另外，下月要要交车险了，又是八百多。幸好一月的工资不抵杂费不扣税，加上会议报销，多少能回点血。

关于州税，听杨书天说，郭紫淳也没有收到退税。听说她打算下次报税时写清情况，让对方一次把税都退来。我报税的时候，也去问问她怎么操作好了。

又忘记了高维统计课的事 .. 犹豫下周还要不要去听。

January 11th, Friday #

周一时看到syllabus，说负责的课有两个助教。除了我以外，还有个下一年级的博士新生。当时很高兴有人能来分担工作，但直到周二，依然没有看到这个助教回复老师的邮件。跟刘心雨打听，得知这个新生不喜欢这门课，正在联系更换课程。周三时老师说下午和新生约了见面，当时觉得插进去聊会把事情弄复杂，就没有参与。

一直担心这个新生一走，我也许会变成唯一的助教。但之前天真地觉得，既然是换课程，总归会换一个别的人过来。如果没有换别人，我就有充分的理由去找工作人员抗议。没想到早上收到老师的邮件，说那个新生被换到tutoring项目了，之前被分到这门课，是工作人员弄错了。轻描淡写了一句之后，下一句马上开始谈其他的工作，要紧事就一笔带过了。

既然是错误，也就无法抗议没有新人被分来了。我也只好小小地回击一句，暗示自己知道实情。除此之外，毫无办法。不得不说，姜还是老的辣。之前以为一个萝卜一个坑，没想到会有tutoring这种可以随意加人手的项目。没有参加讨论，是我太大意了。打算接下来对安排答疑的事，能拖就拖，能缩水就缩水，作为小小的报复。少了一个助教，损失得两个人共同承担 (哼)。

这门课被老师讲得很乱很杂，看不出章法来，也不知从何入手。好在作业是六到十人一组，一共五到七次。虽然是七十人的大班级，大概也吃得消。考试则会放点水了。

January 12th, Saturday #

虽然起床比较晚，且一上午都在收拾东西，但下午还是干了不少活。晚上雷雷约，去学校狼人杀，玩得也很开心。之前担心周末宅家会心情很差，所幸能出来玩一趟。又约了明天中午和雷雷、大姐等等去吃烤鱼。正好可以顺便去买点八宝粥 (???)，过个腊八。

这两天在看漫画“白银之匙”。虽然看漫画本身不太好，但这部漫画还是给了我很多内心的力量。认真做事的人，大概运气都会稍微好一些吧。

January 13th, Sunday #

早上收到Gunter Sharp的邮件，一上来就狮子大开口，问我能不能每周做六小时的office hour。我强压了怒火，回复说我每周最多花一天在这门课上，学生有问题让他们预约吧。昨天还想着跟老人家好好相处，今天老人家就跑来秀下限。之前还不明白为什么另一个助教拼命地要换课程，现在总算明白了。之后老人家要做的烦人事绝对不会少。

转念想想，这也不完全是坏事。一方面，这些闹剧可作为这学期生活中的“鲶鱼”；另一方面，斗智斗勇也有助我的成长。当然，还是要认真应对。鬼知道这老头还能做出什么事来。

January 14th, Monday #

偶尔怀旧，听“易中天品三国”。刚刚听了个开头，讲的是曹操。

觉得评价一个人，最好用矛盾的表述。比如说老李是“阎王脾气”，由于程度、方向都不确定，很难让人准确理解。但如果说老李是“阎王脾气，菩萨心肠”，老李的形象就顿时丰满起来。有人说，鲁智深一辈子“杀人放火，普度众生”，用的是类似的道理。

表述的能力，也代表理解的能力。傅雷谈“热烈又恬静，深刻又朴素，温柔又高傲，微妙又率直”。如果只知其一，不能算知己。网上曾流传某学院的择偶标准，道是“敏而不疑，缜而不繁；娇而不横，求而不奢；诤而有益，谋而有断；处危有度，历难有瞻。”写得这样漂亮，与其拿来择偶，不如用作对自己的要求。

理解别人难，理解自己更难。毛主席说，矫枉必须过正，非过正不能矫枉。但修身路漫漫，方向是总在变的。用矛盾的方法去理解自己，可以避免用力过猛。

January 15th, Tuesday #

连着两天早睡早起，精神比周末时好了许多。决定还是去旁听高位统计课；虽然花时间，但可以为生活增加新鲜感，也可以认识更多的人。只是闭门造车，坚持不了太久的。

但论文又遇到了新的障碍；无数次觉得就要完成了，无数次又遇到新问题，欲哭无泪。但抱怨也不会让文章自己写好，只能重新振作。他强归他强，我自日拱一卒。

还有件事情要反思。早上碰到Yaassin，但脑子里在想事情，竟没注意到。直到他打招呼，我才猛然反应过来。要把注意力集中在当下：走路时要好好看人，聊天时要好好去听。

January 16th, Wednesday #

下午去和Anton讨论论文，不但没能解决已有的问题，还差点产生了一堆新问题 (扶额)。不过，这也是在给老师的讲解中，发现自己先前不足的一个过程；还是大有裨益的。

晚上连续收到三封学生邮件，说明天要来问问题。之前以为可以逃避责任，果然还是想得太轻松了。权衡之下，决定明天上午在办公室准备。因此翘了统计课，和孙浩栋要了统计课的笔记，之后复习。

January 17th, Thursday #

昨天看到吴迪(女)发的中美建交40年大会的消息，说会议在Emory校园举行，可以购票参与。随后和吴迪聊，她说为我弄到了免费票。鉴于大人物集中在周五来，于是决定周五再去。然而早上又收到吴迪的微信，说原来搞错了，免费票只能用于周六一天；于是行程只好作废。

虽然没去成，倒也没什么不满。一方面最近确实事情多，而开会除了长见识，没什么短期的实在用处。另一方面，自己之前有好好地抓住机会，有充分地了解情况，并未因为“怕麻烦”之类的理由逃避。没去成是客观原因造成的，不必纠结于此。

January 18th, Friday #

论文里关于模型Concavity的性质，几番尝试，既无法证明，也无法证伪。折腾三天，非常气馁。

接下来只好避开这个问题，赶在下周三前把论文完成，请Anton先开始修改。

文献综述、Identification、Concavity这些问题，希望之后能再跟上来。

January 19th, Saturday #

约了张焱、小白、海范吃早茶。原以为可以凑一局游戏，但没能成功。于是买了菜就回家了。

久违地在家里做了顿小火锅，吃得心满意足。只是体重并不乐观。

January 20th, Sunday #

周中过得很充实，但周末放肆得厉害。意识到这一点后，决定晚上稍微努力一下论文。

最近做的历史政治方面的笔记，已经渐渐有了点样子。渐入佳境，令人开心。

January 21th, Monday #

早上注意到左侧耳后有点鼓，大概是扁桃体或淋巴结肿。

最近也发现了其他的一些小问题，想必身体已在亚健康状态了，要小心保养。

January 22th, Tuesday #

昨晚没睡好，今天精神有些差。

下午又接到Gunter Sharp的邮件，答复时有些过度反应，老师的回信语气不太妙。接下来还是要认真干活；毕竟对方是老师，要是真惹得他不高兴，怕是要吃不了兜着走。

感觉最近写论文，就像是国民党打淞沪战役；齐装满员的一个师拉上去，几个小时就报废了。我不想继续学老蒋，不打算玩添油战术。而论文是博士四年的核心，也没有绕过去的可能。因此只好全力猛攻，就算花掉一整个学期也在所不惜。兵闻拙速，未睹巧之久也。

因此也只好放弃高位统计课。出尔反尔了，要自责。

January 23th, Wednesday #

之前两天写论文非常用力，有些疲劳。加上睡眠不算好，精神状态不佳，负面情绪也在积累。早上反思，觉得虽然有客观上的问题存在，但负面情绪本身对自己的伤害更大。如果内心足够强大，自己受客观问题的影响就会小得多。另外，精神状态变差也有其他影响。近期健身有些放水，可以作为意志力下降的证据。

赶论文是为了今天拿给Anton看。Anton二月份要去意大利，因此赶在他离开前，要尽量多地完成论文的修改。明天是助教的工作；下午和Anton见面后，今明两天可以调整下作息与状态。

认真工作的感觉很不错。不过作息、健身、读书等等也都是健康生活的柱石，应当同等重视。接下来要努力兼顾这些好习惯，以此栽培积极的心态和意志力，让自己稳定地发展起来。

January 24th, Thursday #

最近在读书笔记里，记了许多关于“战略”和“纲领”的东西。于是昨天下午回家后，也思考了一下自己博士工作的“纲领”。之前没有细想过研究要怎么做，只是单纯地打算碰到什么做什么。但如果想找到好教职，就必须要做出有既深度、又有热度的研究，要写出核心论文来。

仔细研究过后，决定选择“网络定价与优化”这个方向做下去，以兼顾热点、深度和可行性。原计划接下来要写的“高维定价”问题，可以朝着这个方向扩展，来冲击年底的INFORMS。而和网络优化关系较弱的研究，无论是否已经做好准备，都要暂时下调优先级。

January 25th, Friday #

如何减少负面情绪，其实还是个注意力的问题。之前情绪低落时的反思，大多在于“后悔做了什么”或者“以后不该做什么”。然而，比起“不该做什么”，更加重要的问题是“该做什么”。修身在于要有“清源活水淌进心来”。一个人，如果越发地朝气蓬勃，精神旺盛，能吃能睡，则绝大多数困难都会大事化小、小事化了。

January 26th, Saturday #

中午和海范等人吃饭，吃到了久违的油条、豆腐脑、羊肉泡馍。下午帮他搬家，对他的新家很羡慕：面积很大，结构合理；最重要的是，阳光十分充足。当然，价格也很丰满，尤其对于我这样的穷学生 (笑)。

January 31th, Thursday #

之前按照“格物致知、正心修身、齐家治国平天下”的顺序，把读书笔记分成了很多层次。格物讲的是美学，致知讲的是哲学，正心讲的是心理学，修身讲的是相处之道，齐家讲的是管理学，治国讲的是谋略，而平天下讲的是技术和制度。这理解大概与古人有所不同，是一家之言。

美学、哲学，暂时不太顾得上；等到要看的时候，“文学回忆录”和“苏菲的世界”可以作为起点。正心、修身，主要看的是“幸福课”、“亲密关系”两门公开课。至于学习齐家、治国、平天下，则主要是靠读历史。

其实在“亲密关系”和“管理学”之间，还少了一个环节：如何开启更多的关系。拓展人际网络，认识新的朋友，始终是我非常想做到，却又不知如何做的事。唯独这方面，完全想不出有什么好书可以读。如果之后能打通这一环节，整个系统会更加畅通起来吧。

之前跑步，常常跑不了多久就开始膝盖痛、脚踝痛。虽然觉得是自己姿势不对，但却一直找不到正确的姿势。网上说跑步要脚掌先着地；操作起来，很难做到。鞋跟太厚也许是其中的一个原因。

昨天看到一个Ted演讲，说光脚跑步可以“强迫”自己用正确的姿势。今早趁着人少，去健身房尝试了一下。姿势确实有改观，但脚被磨得够呛。穿很软的薄底鞋也许更好一点。

“流浪地球”要在亚特兰大上映了，打算周末去看，还在努力约人中。

February 7th, Thursday #

上午批作业，下午答疑，忙碌一整天。批作业比较痛苦，因为连蒙带骗，一心只想快点做完。相比之下，答疑要快乐些。课程的内容还是要认真学的；能回应学生的期待，才会觉得快乐。

气候骤暖，晚饭吃得饱，就出门散步。人家遛狗，我遛自己，也觉得愉快。久违地在傍晚遛弯，触动许多回忆，五味杂陈。（如和张焱等玩Pokemon时、带梦莹来家里时，都是像这样地在小区里走过。）说五味杂陈，是因为过去那些小小的幸福，都悄然从指缝中溜走了。时间过去，我依然一次次寻找着热闹，又一次次回归孤独，不曾知道要如何安放自己的内心。

我想，好的文学家，都该曾细细品尝过幸福、孤独。因此才能寥寥数语，就把读者带进某种情绪里。

对我而言，无从安放的内心，是我为了改变自己去祈祷、去努力的原因。而改变，也确实渐渐地发生在我的身上。失去本身虽然不可阻挡，但若能多创造些幸福、多抓住些幸福，也就会少些后悔吧。

效率很低地度过了一天。回家觉得至少要健身，所以努力练了腿。

觉得幸福感有三大支柱，作息规律、饮食健康、经常锻炼是一个，目标明确、努力工作、保持专注是一个，有交往很深、经常相处的朋友又是一个，缺一不可。而我一直一瘸一拐。

“流浪地球”不负期待。虽然小漏洞很多，但不难看出制作者的用心。地球发动机，帅气得不行。

February 10th, Sunday #

二月已过去十天。十天莫名地过去了，论文却进展甚微，内心非常痛苦。然而越是痛苦焦虑，就越是无法集中起精神，也就更加气馁。这时如果是在学校，情况就会好些。自己在家，总是停不下来地胡思乱想。

以前遇到这种情况，就会靠刷漫画打发时间，自欺欺人。但这次已经决定戒掉漫画，于是只好无所事事、无所适从地发呆。好像爬悬崖爬到一半，努力维持自己不掉下去，但也没办法继续往上爬。

刚刚看到一篇关于抑郁症的文章。作者说自己抑郁，自罪自罚；但朋友很多，并不孤独。我倒不觉得自己抑郁，但确实很孤独，因此内心常常无法平静、专注。细想之下，又觉得着不仅是多交朋友的问题。海范、于寰生活里的娱乐活动，似乎不比我多。但两个人都可以长时间、专注地工作，而不需要靠娱乐来刺激自己。

要怎么办呢。

February 11th, Monday #

前天下午健身之前，打算喝点牛奶。第一口喝下去，才发现已经坏了。晚上拉肚子，吃了点药，就没再在意。昨天下午忽然头疼，而且完全没了食欲，晚饭只吃了几口。好容易熬到晚上，早早睡了。

很担心今天情况会更严重，但早起后一切正常，感觉还不错。早餐只喝了点牛奶，小心翼翼，不敢掉以轻心。接下来得好好工作，努力保养；免得身体出状况了才知道着急。

昨晚看“邓小平时代”，心潮澎湃，很晚才睡着。但幸而早上自然醒，体力也不算糟。

雨后清晨，温暖而湿润，心情大好。

February 14th, Thursday #

昨天和叶姝文在微信上聊了许多。自五年半 (卧槽好快) 之前伯克利之行结束后，就只在两年前和她聊过几句。那时她五年本科加两年硕士即将结束，正在找工作。而我则是一边准备实习，一边等着博士入学。

通过聊天，想起了不少已经淡忘的事。叶姝文当时和王晓雪、姚敬先、张启翔一起租了一个房子，而姚敬先的女朋友庞小月另找了房子住。伯克利的暑期项目，我是自己找的，而他们则是跟学校的团。记得当时还有刘可和舒畅，后来还认识过孙思濛。但和所有这些人是怎么认识的，我完全不记得。唯独记得因为没有跟大部队去购物，所以和杨文清一起逛了三番。当时她帮我拍过几张照片，我非常喜欢。

那个暑假具体的经历，也大多只剩下片段的记忆。唯独翘课去东边看学校的事，还有较为深刻的印象。

February 15th, Friday #

早上在想伯克利那个暑假的事。回想起来，当时也许是我前二十多年生命中，一个小小的高光时刻。虽然浑身都滋滋冒着傻气，但胆子大得不行，满心光明。当时做的许多傻事，都成了我改变的契机。

记得当时写了些文章和日记，打算之后翻出来看看。

现在的自己和当时比，是不是更加接近梦想了呢 (笑)？

February 16th, Saturday #

前几周把王立群的“秦始皇”听了十几集。觉得春秋战国时期的历史，主要是有趣，各方面都很像是“权力的游戏”。当然，春秋战国也是各种思想形成的时候，能展现中国历史发展的一些脉络。但这段历史，一来史料极为有限，二来斗争水平还没有全面地提上去，所以能摘抄的东西不算太多。

前段时间主要是在读三国史。易中天的“品三国”按顺序听了下来，只剩几集了。王夫之的“读通鉴论”里，写献帝时期和三国时期的各有一章，我稍微浏览了几遍，做了些摘抄。三国志跳着看了一点，也摘抄了一点，用来当“品三国”的笔记。资治通鉴里有关三国的部分，我还没读，暂时也不打算去读。吕思勉、陈寅恪先生有关三国的书，暂时也不打算去看。一下子看得太多、太深入，还是要厌倦的。

有关三国以前的历史，等听完王立群、易中天的视频后，就暂时作个小结，休息、沉淀一下。作为总结和沉淀，打算写个系列小短文：“诸葛先生的演讲集”。过程中会跳着读一些毛选，以便模仿主席的口气。

最近挖了两个新坑。其中一个是熊向晖的几本回忆录，里面讲了自己做间谍的亲身经历，也讲了周恩来做情报工作的一些有趣方法。周恩来“闲棋冷子”式的布局思路，我非常喜欢。这本书可以作为“用间篇”读书笔记的资料，也可以作为对建设人际关系的一些指导。近期先依托它来建立思维框架，细节之后再补充。人际关系方面有很多杂书，以后可以找点时间去粗取精。相信这个过程会改善我的交流习惯。

另一个新坑是傅高义的“邓小平时代”。之前的三国史笔记，主要在讲框架性的思路。要想从具体的案例中进一步地学习，大概很少有比这本书更好的资料了。暂时打算把这个放在熊向晖回忆录的后面。

February 17th, Sunday #

体重反反复复。明明觉得最近两天吃饭、锻炼都很用心，但还是变重了不少。想了想觉得，一直以来体重的上上下下，其实主要是水分和肌肉量的变化。因此生活节奏一变好（喝水更多、锻炼更多），体重就不减反增。而整个过程中，脂肪量的变化其实很小。

因此，接下来要每天努力喝水、锻炼，长期保持好的生活节奏，以清晰化脂肪的减少。吃东西方面，因为和老陈每天一起吃，所以饮食结构暂时还没办法大改。但短期里可以养成细嚼慢咽、八分饱的习惯。

也不要被一时的得失影响了情绪，要从全局的角度想问题。东边不亮西边亮，总有可以前进的地方。

February 18th, Monday #

又和叶姝文聊天，聊到卡坦岛，于是去买了一套。之前老陈提过要买，但一直没买。把之前买的三国杀也整理了一下。之后可以找机会约人玩。心想如果以后自己买房安家，一定要弄一个桌游角，摆些地毯、茶几、垫子、懒人沙发。每逢假期，就可以约几个小伙伴来家里玩。

February 19th, Tuesday #

昨天和今天，花在工作上的时间都不少，但效率不高。虽然花时间思考是对的，但不能光是一直想，而是应该一点点往下写，稳步地向前推进。另外，还是要注意休息。一直想个不停，越往后就效率越低。

February 20th, Wednesday #

去做了那个“耸肩循环”，比想象中的累很多。

和杉树那边确认了入职，租房也开始找了。这边的房子准备提前退。

不知道以后能不能找到处得来的新的室友。

February 21th, Thursday #

花了整整两天批作业，而之后两周还有两次作业要批。最近实在花太多时间在助教的事情上了。

也许是因为这个，今天脾气非常暴躁。

决定故意拖延工作，免得老师没完没了地给我找事做。

February 22th, Friday #

之前下载了电脑版的“卡坦岛”，沉迷了两天；今天起要收收心。不过打算联系点朋友，一起玩桌游版，以此增加感情、打开社交局面。回想以前的小伙伴，如淳映、子旋等，都是乐于张罗活动、介绍自己的朋友互相认识、分享自己社交圈的人。如今身边这样的朋友太少，就只好自己出来做这样的人了。

关于批好的作业，最后决定还是不拖时间了。不想让自己变得太计较，太难相处。

这几天也看了很多健身视频，只是健身反倒耽搁了。接下来要努力锻炼。

February 23th, Saturday #

昨晚和家里视频，老妈批评我在人际交往方面不主动，连自己家兄弟姐妹都漠不关心。睡前发现当天是马思爽生日；去说了句生日快乐，才得知她原来已经结婚很久了。又想到半年来大家都要离开；好像我一直只看着自己，一抬头发现大家都变化了那么多，就觉得非常伤心。早上起来，看到大姐他们去看画家Mengxi展览的照片，又是双重打击。

大概是性格使然吧。之前想象末世时我会怎么做，想到的也是做独狼。但我其实并做不来独狼，这就是我性格矛盾的地方。想来想去，还是要变得用心和主动。

早上和千千聊天，听下来她的规划，就建议她来美国考CPA。“路子野一点，越级打怪，才能改变命运” —— 我这么和她说。而对于目前的我而言，越级打怪的机会在哪里呢？要好好想想。

February 24th, Sunday #

周五、周六两天晚上睡得迟，尤其是周六。两个晚上都因为聊天而比较兴奋，进而失眠。因此，周六、周日两个上午都无所事事地度过了。按计划，周六一整天用来娱乐，所以无妨。但周日就损失掉了。

浪费了上午后，觉得至少下午要做点什么。于是收拾屋子、健身。晚上不情不愿地交了这半年的车保。与上次相比，贵了将近一百块，应当是杀熟无误了。之前就是因为StateFarm的人毫无底线地杀熟，才转到Progressive上来，然而天下乌鸦一般黑。问了StateFarm现在的报价，要便宜一百块；但要重新上安全驾驶课，实在麻烦，所以暂时不换，下次再看。

前两天觉得要主动做人际的交往，这两天就有了效果。昨天下午去和博士同学打卡坦岛，以此认识、熟悉；晚上去白伟成家里吃饭，饭后和海范、高幸、老陈一起聊天。下周打算约雷雷、大姐打卡坦岛、爬山。希望这样的趋势能继续，希望自己能成为熊子旋那样的，朋友圈子里的“关键人物”。

February 25th, Monday #

昨天今天阳光很好，一扫之前阴雨连绵的阴霾。虽然早上冷，但中午过后就十分温暖。这样的天气却呆在屋子里，实在是莫大的罪过。好在论文进展不少，可以自我安慰。

February 26th, Tuesday #

和小区物业说了提前退掉现在的公寓，也找好了上海的房子。接下来要找个之后的室友，但总觉得希望不大。自己住不仅是贵，实在是很孤独。之前自己住的时候，总是有点怕回家。好处倒也不是没有：自己做饭因此利于减肥，方便自由，可以随意领人回家 (前提是有人可领)。目前而言，走一步看一步了。

周三上午去学校听了seminar，然后把文章弄好发给Anton。之后到周五中午的两天，都在看杂书、写网站，算是今年和Anton最后一次meeting之后的放松。

周五下午陪雨婕，晚上和马哥、聪哥、雷雷、杨书天吃饭。今早和家里视频，然后去Costco、HMart买菜；下午陪雨婕去看公寓，签了合同，然后晚上又去和小白、海范吃饭。一不小心，“无事忙”了两天。

需要尽快计划一下论文、面试准备、投资算法等等这些事，都要如何安排。

December 17th, Tuesday #

周日和周二都是一整天的杂事，打网球、约饭、买菜、申请实习等等。周一则是在放羊。

Wi-Fi一直在出各种问题，有点心烦。

December 18th, Wednesday #

开始学C++，但感到很头大，一上来就遇到问题。

下午找海范、陈鹏打羽毛球，顺便问了海范C++的问题，勉强解决。

也再次开始做交易策略的研究，慢慢地编程。

December 19th, Thursday #

时间过得飞快，转眼离回国只剩下一周。上周后半都在放羊，这周才慢慢开始干活，只是效率不高。

当务之急是做好Sabre的面试准备，希望能在回国之前完成。

December 20th, Friday #

早起后工作六小时，然后锻炼；久违地高效了一天。一直在看Sabre的面试题，越看越有趣。

本来约了于寰和海范吃饭，但时间太紧，改到了明天。晚上和徐晨妍等一众人去看Light Show。

December 23th, Monday #

渐渐进入节日状态，娱乐活动一件接一件，干活就搁置下来。周六上午在工作，中午去和海范去找于寰吃饭，然后晚上剧本杀。周日白天都在陪雨婕瞎折腾，晚上则在家和陈鹏打游戏。今天上午去Costco买了些带回国的东西，下午收拾屋子；晚上约了雨婕来打Switch游戏。

December 24th, Tuesday #

昨晚二姨忽然打来电话，边哭边说，老爸十月份被查出肠癌，正在化疗，一直瞒着我。

于是赶紧向二姨要来诊断书，去网上查了一下，又给二姨夫、老妈打了电话，详细问过情况。得知化疗情况很好、之后要观察是否做手术后，心理稍稍松了口气；但还是止不住地担心，生怕有什么变数。

失眠半宿，今天一上午都昏昏沉沉，勉强洗漱过吃了中饭。陈鹏说下午去打网球，叫了徐晨妍和赵菲一起。刚打时还好，但越来越没办法集中精神，只是一直丢球。陈鹏和徐晨妍问了几次，我只说家人身体不好。

和家人打过电话、躺在床上想事情时，忽然决定要和雨婕分开。起因是觉得雨婕简直灾星，整日叨逼叨自己姥爷得癌症后如何如何；结果半年来，我家人倒被查出癌症了。刘叔急性胃癌去世了；姥爷得胆囊癌，年纪太大只能静养；如今老爸也被查出肠癌。想来想去，天旋地转，怒火中烧。

等到冷静下来，觉得不该拿这种事去怪雨婕；但也下了分开的决心。自己已经是这个年纪，却为了些低级的欲望去做这种过家家的事，耽误两个人的时间，实在有些不应该。还是该认真地找人生伴侣吧。

于是上午和雨婕直说了，顺便退了那些QQ群，决定不再做这种“网约SM”的事；希望能以此为老爸祈福。

晚上一群人来家里聚餐、玩剧本杀。平安夜，愿自己和家人都平安。

December 26th, Thursday #

颓废两天。昨天下午和晚上找人来家里打麻将、吃饭、玩游戏；今天则是被陈鹏和Rosy拉出去遛狗、看各家各户挂的彩灯。其中零碎的闲暇时间，都在看一本叫“非人哉”的小漫画。

本来答应要帮千千和小姨买包，却错过了能网购的时间，很愧疚；打算明天上午去outlets试试运气。等到逛完街回来，就该收拾回国的东西、准备电话卡和VPN了。

以往回国都很开心，这次更多的是忧虑。希望能一切顺利。

December 27th, Friday #

早上忽然发现，机票是明天凌晨的，而不是中午。幸好发现了，也幸好之前没有强行网购东西。上午去outlets逛，小姨和千千要的东西都买到了，算是可以放心。中午吃过饭，陪陈鹏去Cumberland Mall看了下苹果手表，然后就回家补觉。醒来后开始整理行李，准备去机场。

December 31th, Tuesday #

周五晚上从亚特兰大出发，到延吉已是国内的周日上午。因为时差，周日、周一都过得有些乱。期间和爸妈视频，商量了下一步的行程；也和二姨夫、二姨、小姨沟通了家里的事。

老爸前面一阶段的化疗已经结束，评估结果今天出来。本来如果情况尚可，接下来就要决定怎么做手术了。但今天拿到的结果却说，老爸肺部有阴影，不能确定是癌症转移还是其他的疾病。二姨夫说，这些还都没和老爸老妈讲，想等结果明确了再看。听到这里，心一下子又揪起来。

早上二姨说，想给老妈拿十万块钱，但怕她不要；这才担心家里还有多少存款。

普通人的普通家庭，在这样的疾病面前，真的挺渺小的。

昨天陪姥姥去交水费，路上聊到刘金善叔的事。姥姥说心里觉得这人大概已经没了，但家里都瞒着她；又说她问小姨时，小姨歇斯底里地哭过。听到这里，心理不是滋味。

昨晚听二姨夫说，十月份姥爷病重，趴在床上下不来，粪便流了一身。没有办法，只能叫救护车来用担架抬下楼。揪心之余，想到姥爷现在身体情况很好，欣慰了很多。又想到这两天自己还在让姥爷做饭，忽然很愧疚。即使现在看上去健康，但还剩下多少时间，大家心里都清楚。

旅途劳顿；这两天又出门几次，有些着凉。今早起来，感冒忽然汹涌而来；赶紧吃了药、喝了维生素，然后静养。希望感冒能尽快好吧；一方面很快就要赶路，一方面万一传染了家人，可不是闹着玩的。

转眼又一年；这一年的末尾，不想以往那样顺利。

周生春教授学术论文集


简介

周生春教授出生于1947年8月29日，祖籍江苏淮安，因父母工作原因迁居苏州。淮安周氏诗书传家，周老师幼承庭训，酷爱读书，在动荡的十年，积累了深厚的学养。1978年国家恢复招收研究生后，周老师考入杭州大学历史系，师从陈乐素、徐规先生攻读宋史方向的硕士研究生。1981年硕士毕业后，曾暂时到苏州大学工作。1983年又考入北京大学历史系攻读博士研究生，师从邓广铭先生继续研究宋史。1986年完成博士学业后到浙江大学任教，先后任职于哲学系、哲学社会学系、经济学院等，是一位富有人文关怀的历史学者。周老师研究兴趣广泛，涉及考据、经济史、思想史等多个领域。虽然著述不算丰富，但所发表及出版的著作，皆属精品。

周老师发表的论文存在三种情况：一是独立发表，二者作为第一作者发表，三是作为第二作者发表。根据我们初步统计，周老师独立发表在学术期刊及收录在各类论文集中的论文，约有47篇。周老师作为第一作者发表的论文复存在三种情况，第一是主要由周老师撰写，第二作者挂名学生；第二是主要由学生撰写，第一作者挂名周老师；第三是由周老师与合作者分工完成。前面两种情况，主要是为解决学生的毕业问题，由浙江大学硕、博研究生的培养规定导致。我们坚持严格筛选的原则，论文集只收录主要由周老师撰写的论文，周老师与合作者分工撰写的论文设为附录，而凡是主要由学生撰写以及周老师为第二作者的论文，一概不收录。宁缺毋滥，旨在向大家呈现周老师的学术精华。

从内容来看，周老师的研究主要集中在三个学术领域，即文献学、经济史和思想史，还有几篇似可列为杂论。周老师的文献学方法秉承陈垣先生，考据工夫十分深厚，在这一论域的著述也相对丰富，从1978年至2012年的三十多年间，先后在《历史研究》、《文献》、《文史》、《浙江大学学报》等重要学术期刊及专业论文集上发表论文23篇，尤其他精心辑校的《吴越春秋辑校汇考》一书，1997年由上海古籍出版社出版以来，得到学术界持续好评。2019年，该书又由中华书局再版。从2014年开始，周老师还参加了“浙江文献集成”项目，指导学生重新整理《越绝书》、《吴越春秋》、《吴地记》、《古越书》等。

周老师研究的第二个领域是宋代经济史，尤其是宋代江南区域经济史与两浙文化，这是他读硕、博研究生时期确定的研究方向。从1991年到2010年的二十余年间，他先后在《文史》、《中国史研究》、《浙江大学学报》、《中国农史》以及相关会议论文集、纪念文集发表学术论文约17篇，与学生合作发表2篇（附录1），多为该领域的研究所引用。另外，周老师还与李伯重教授共同主编的《江南的城市工业与地方文化：960-1850》、独立主编《“印刷与市场”国际会议论文集》等，都在宋代经济史研究中产生了重要影响。目前，花木兰出版社正在编辑他的最后一部著作《宋元江南农业经济史论稿—以浙西、江东水利田的开发为中心》——由他的博士论文修改而成，将于10月份正式出版，相信亦必嘉惠学林。

周老师研究的第三个领域是中国古代思想史，约有论文11篇。另外，还有与他人合作的思想史论文2篇（附录1）。他早年倾心道家，1990年代以来，独立及与他人合作在《哲学研究》、《浙江大学学报》、《自然辩证法研究》等期刊上发表多篇有关老子研究的学术论文，并先后撰成《白话老子（帛书）》（三秦出版社）、《老子注译》（太白文艺出版社）两书，后一书又修订为《老子注评》由凤凰出版社再版。1990年代后期，他在思想史领域的学术兴趣逐步转向儒家，又独立及与学生合作在《哲学研究》、《中国学术》、《中国经济史研究》等刊物上发表多篇有关儒家、儒商的论文，都得到了比较高的学术评价。2006年与杜维明教授一起创办浙江大学儒商与东亚文明研究中心，开展经典会读活动，整理出版“经典会读”系列丛书——《论语》、《大学·中庸》、《孟子》、《老子》等。2008年担任《儒藏》（精华本）的编委，指导学生一起整理《和靖尹先生文集》及其《论语解》、《孟子解》。前者已由《儒藏》项目统一出版，后两种正由浙江大学出版社排版。2009年，周老师主持澳门企业家研究项目，项目报告《儒家价值观与现代经营管理——澳门华人企业家访谈录》也已整理成书，正在由澳门三联出版社排版。

除了上述论著，周老师的研究还涉及古代监察制度、宋代两浙文化、宋金关系以及辽、金、宋三史编撰等问题，独立或与学生合作在各类刊物和论文集发表论文约8篇。并积极探讨中国传统学术的现代转型，编撰出版《中华传统学术的现代转型：以中医为例》一书。涉猎中国审计文化、行政管理等课题，与陈正兴合编《中国审计文化研究》，与陈倩倩、汪杰贵合著《韩国地方政府管理》等。此外，周老师还参加了《中国史学名著评介》（仓修良主编）、《中国事典》（中外名人研究中心编）的编撰工作，撰写了“宋史”、“金遣高忠建等讨伐窝斡”等词条，合计约4.6万字。

本论文集共搜集周生春教授各类论文约66篇，计划分为五编，即考证、经济、思想、杂论和附录。我们计划将周老师撰写的《中国史学名著评介》“宋史”、《中国事典》“金遣高忠建等讨伐窝斡”等词条，亦放入附录。词条合计约4.6万字，我们预估各部分论文约36万多字，估计全书在40万字以上。详细论文信息见论文集目录。
